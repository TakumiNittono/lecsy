# Founder Mode — 委譲しない5領域

> 出典: [[NotebookLM_学習/② YC創業者マインド]] / Garry Tan, Paul Graham

## 原則

Paul Graham が提唱した Founder Mode の核心:
**「Manager Mode（経営に徹する）」に退避すると会社は堕落する**。
創業者は細部に踏み込み続けるべき領域がある。

## 委譲しない5領域

### 1. ユーザーへの共感と「味付け（Taste）」
- ユーザーが何を求めているか、製品がどうあるべきかという**「感性」は他人に委譲できない**
- 特に B2B では、**顧客がどうすれば昇進できるか**まで理解する深い洞察が必要

### 2. 詳細へのこだわりと直接の関与
- 経営が軌道に乗っても、現場から離れて空の上から指示を出すのではなく、
- **重要な詳細部分には自ら関与し続ける**

### 3. 採用と文化の構築
- 誰をチームに入れるかは、**そのまま「会社が何であるか」を決定**する
- 初期の採用において、自分のビジョンを熱狂的に伝えられるのは創業者だけ

### 4. ビジョンの伝道（エバンジェリズム）
- 資金調達、営業、広報など、あらゆる場面で世界を熱狂させる
- **「チーフ・エバンジェリスト」としての役割は創業者が担うべき**

### 5. 説明責任（Accountability）の保持
- 自分が細部に注意を払っていることを**周囲が知っているだけで**
- 組織には魔法のような変化が起き、ミッション達成の確率が高まる

## lecsy 的実践

- **委譲していいもの**: 経理、LP のコピー編集、バックエンドの細かいリファクタ
- **委譲してはいけないもの**:
  - パイロット校との商談（Ryoko との関係）
  - UI の最終決定（特に録音ボタン周り）
  - 最初の10-20人のハイヤー（いずれ）
  - 週次ブログ/投稿の執筆
