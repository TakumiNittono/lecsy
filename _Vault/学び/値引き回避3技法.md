# 値引き回避3技法

> 出典: [[NotebookLM_学習/③ 価格ポジショニング]] / Chris Do, Chris Orlob

## 原則

値引き交渉を防ぐには、**会話の主導権を握り、価格を「コスト」ではなく「投資」として認識させる**。

## 技法1: 「処方の前の診断」を徹底する

顧客が「機能」を求めてきても、**即座に提案しない**。

質問例:
- 「なぜ今それが必要なのですか？」
- 「解決しないと**年間でいくらの損失**ですか？」
- 「過去にこの問題でどれほどの機会損失がありましたか？」

→ 問題の大きさを**顧客自身の口から語らせる**。

## 技法2: アンカリング (Anchoring)

具体的な見積もりを**提示する前に**、解決によって得られる経済的価値を会話に登場させる。

**悪い順序**:
> 「月額 $500 です」 → 「高い！」

**良い順序**:
> 「年間 $40k の売上増の可能性があります」 → 「月額 $500 です」 → 「安い！」

**最初の大きな数字が基準（アンカー）となり、その後の料金が相対的に安く感じる。**

## 技法3: 提案書を送る前に「口頭で」合意を取る

詳細な見積書を送りつけるのではなく、**事前に確認**:

> 「この課題を解決するために**月額 $500 程度**かかるとしたら、
> 進める価値はありますか？」

- YES なら → 正式な見積書を送る
- NO なら → なぜか聞く（価格ではなく価値認識の問題を特定）

**顧客が納得していない段階で見積書を送ると、項目ごとに細かく削られる対象になる。**

## lecsy の営業フロー

1. コールドコール / メール → 商談獲得
2. **ディスカバリー**（[[ディスカバリー質問リスト10]]）で問題の大きさを数字で聞く
3. **アンカリング**: 年間売上増の金額を先に提示
4. **口頭合意**: 「月額 X なら進める価値ある?」
5. YES → 正式見積 → 契約
