# Bootstrap Founder 日課5 — 成功者の毎日

> 出典: [[NotebookLM_学習/⑤ インディーSaaS]] / Pat Walls, Arvid Kahl, Courtland Allen

## 1. 「書く」習慣
- **毎日ブログやジャーナルを書く**ことで思考を整理し、ビジネスの優先順位を明確に
- Pat Walls は **1年半、毎日書き続けた**
- lecsy 実践: 毎朝15分、開発ジャーナルを書く（Obsidian の日次ノート）

## 2. 2時間の「ディープ・ワーク」
- 本業（OPT の仕事等）が始まる前に、**最も重要な開発や執筆**に2時間を充てる
- 通知 OFF、メール 閉じる、1タスクだけ
- lecsy 実践: **朝6:00-8:00** をディープ・ワーク固定化

## 3. 顧客との直接対話
- **毎週**、新規ユーザーや見込み客と直接話す
- 彼らの **「Aha 体験（価値を感じる瞬間）」**がどこにあるかを確認
- lecsy 実践: 毎週 **最低2件の 30分 Zoom**（パイロット校 + 新規見込み）

## 4. プロセスの自動化・文書化
- 自分が動かなくても回るよう、**サポートや管理業務を可能な限り自動化・マニュアル化**
- **「売却可能なビジネス」を作る意識**
- lecsy 実践: 月末の**処理フローを毎回 Loom 録画** → 将来の外注マニュアル

## 5. 意図的な休息と運動
- バーンアウトを防ぐため、**テニスやサイクリング、家族との時間**など仕事以外のルーチンを死守
- lecsy 実践: **週3回ジム** + 土曜は完全オフ

```
06:00 起床

08:15-09:00 朝食・散歩
09:00-17:00 OPT 本業
18:00-19:00 ユーザー対話 or ジム（習慣3 or 5）
19:00-20:00 自動化・ドキュメント（習慣4）
20:00-22:00 家族時間・休息（習慣5）
```
