# Start With Why — lecsy ピッチへの適用

> 出典: [[NotebookLM_学習/⑫ 交渉術]] / Simon Sinek

## ゴールデンサークル

```
    ┌───── WHY (なぜ) ─────┐
    │                      │
    │  ┌── HOW (どう) ──┐  │
    │  │                │  │
    │  │  ┌─ WHAT ─┐   │  │
    │  │  │ (何を) │   │  │
    │  │  └────────┘   │  │
    │  └────────────────┘  │
    └──────────────────────┘
```

## 原則

**機能（What）**ではなく、**存在理由（Why）**から伝える。
→ 意思決定を司る**脳の大脳辺縁系**に直接訴えかける。

## 悪い例（What から）

> 「lecsy は、AI 文字起こしと自動要約ができる、語学学校向けSaaSです」

## 良い例（Why から）

> **「lecsy は、言語の壁を壊し、世界中の学習者の可能性を解放するために存在します。**
> **我々は、教師が書類仕事から解放され、生徒ともっと人間的に向き合える世界を信じています。**
> その信念を形にするために、AI 文字起こしと自動要約機能を持つ SaaS を語学学校向けに開発しています」

## lecsy の Why の候補

### 候補 1: 言語の解放
> 「言語の壁を壊し、留学生の可能性を世界へ広げる」

### 候補 2: 教師の解放
> 「教師が書類仕事から解放され、生徒と人間的に向き合える時間を取り戻す」

### 候補 3: 認識の変革
> 「語学教育を『非効率な事務仕事の集合』から『成長を測れる科学』に変える」

## 使い所

- **コールドコール冒頭**（20秒の後半）
- **メールの1行目**
- **商談の最初の2分**
- **ピッチイベントの導入**
- **LP のヒーローコピー**

## テスト方法

1文の Why を作って10人のターゲットに読んでもらい、**感情的反応**を見る:
- 「ワクワクする」→ OK
- 「なるほど」→ まだ理屈。もっと感情に訴える
- 「よくわからない」→ 抽象的すぎる。もっと具体
