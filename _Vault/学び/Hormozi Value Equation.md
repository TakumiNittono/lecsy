# Hormozi $100M Offers Value Equation

> 出典: [[NotebookLM_学習/① 米国B2B営業]] / Alex Hormozi

## 方程式

```
                  Dream Outcome  ×  Perceived Likelihood
  Value (価値)  = ─────────────────────────────────────────
                  Time Delay     ×  Effort & Sacrifice
```

分子が大きく、分母が小さいほど「**断るのが馬鹿らしくなるオファー**」になる。

## 4要素

| 要素 | 意味 | レバー |
|---|---|---|
| **Dream Outcome** | 顧客が達成したい究極の目標 | 大きな未来を提示 |
| **Perceived Likelihood** | 「本当にできるか？」の信頼性 | 実績/保証/リスクリバーサル |
| **Time Delay** | 結果が出るまでのスピード | 最短化 |
| **Effort & Sacrifice** | 顧客側の作業・苦痛 | Done-for-you で最小化 |

## lecsy に当てはめたオファー例

- **Dream Outcome**: 「留学生の入学プロセスを100%デジタル化し、**入学率を1.5倍に**」
- **Likelihood**: 「同様の州立大学付属語学学校での**導入実績と、24時間体制のサポート**」
- **Time Delay**: 「導入決定から**最短48時間で、最初の学生願書を受付可能**」
- **Effort & Sacrifice**: 「**Done-for-you モデル**: 既存 Excel データの移行、全スタッフのトレーニング、HPへのフォーム埋め込みは**すべて lecsy 側で代行**。貴校の作業はゼロ」

## クロージング文

> 「このオファーを断ると、貴校は**事務時間を週10時間浪費し続け、学生の入学辞退を防げず、年間 $40k 相当を失い続ける**ことになります。
> 始める理由は48時間以内に最初の願書を受け付けられること。
> 始めない理由はありますか？」
