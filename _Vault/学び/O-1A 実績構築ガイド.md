# O-1A 実績構築ガイド

> 出典: [[NotebookLM_学習/⑬ OPT 移民]] / Sophie Alcorn (Alcorn Immigration Law)
> ⚠️ **免責**: 一般情報のみ。個別判断は移民弁護士確認必須

## O-1A とは

**Extraordinary Ability（卓越した能力）**を持つ外国人向けビザ。
- 抽選なし
- 創業者にも取得可能
- 3年 + 延長可能

## 証拠の8つの基準（3つ以上クリア）

| # | 基準 | lecsy 的証拠例 |
|---|---|---|
| 1 | **国際的・国内的賞の受賞** | Y Combinator, Techstars, DevAwards 等 |
| 2 | **権威ある会員資格** | ACM, IEEE, 学会フェロー等 |
| 3 | **メディア掲載** | TechCrunch, The Verge, 日経等 |
| 4 | **審査員経験** | ハッカソン審査、雑誌査読 |
| 5 | **独創的な貢献** | オープンソース、論文、特許 |
| 6 | **専門誌への寄稿** | Medium publication、学会誌 |
| 7 | **展示会での出品** | SIGGRAPH、Cannes Lions 等 |
| 8 | **高額な報酬** | 業界平均を超える給与、売上 |

## 準備期間

専門家（Sophie Alcorn）によれば、**戦略的に実績を積み上げることで 6〜9ヶ月程度**で資格を満たすことが可能。

## lecsy 的 Evidence Building 計画（6ヶ月）

### 月1-2: 基盤作り
- [ ] オープンソースプロジェクト（lecsy の一部 or 独立ツール）を GitHub 公開
- [ ] Medium で「Building lecsy」シリーズ開始（月2本）
- [ ] LinkedIn で専門性を発信（[[Justin Welsh LinkedIn 3段階システム]]）

### 月3-4: メディア露出
- [ ] TechCrunch / Forbes / Japan Times に**寄稿記事**を送る（拒否されるのが普通、毎月1回）
- [ ] Podcast ゲスト出演を狙う（Indie Hackers, 日本の技術系Podcast）
- [ ] ハッカソンの**審査員**に応募（大学系、起業家コミュニティ系）

### 月5-6: 権威の獲得
- [ ] 学会発表（ACL、EMNLP の workshop 等）
- [ ] B2B 賞に応募（EdTech Awards, ASU+GSV 等）
- [ ] 業界団体（IEEE、TESOL 等）の専門委員に申請

### 継続
- [ ] ユーザー数、収益、導入校数などの**数値的実績**を蓄積
- [ ] 推薦状を書いてくれる専門家を6名確保（業界有識者）

## エージェントモデルの活用

俳優やモデルのように**「エージェント」を通じて O-1 を取得**すれば、
自分の会社だけでなく、**複数のクライアントからの収入（コンサルティング等）**を得ることも可能。
