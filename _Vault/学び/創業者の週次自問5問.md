# 創業者の週次自問5問

> 出典: [[NotebookLM_学習/② YC創業者マインド]] / YC 諸家
> 推奨: **毎週金曜の15:00に10分間**、静かにこの5問を自問

## 1. 口コミは生まれたか？

> 「今週、人々が私の製品を**友人に自発的に勧めるほどの感動**を与えたか？」

口コミこそが成長の唯一の指標。YES でなければ PMF に遠い。

## 2. 不快なタスクを最初にやったか？

> 「ToDo リストの中で、私が**最も不快だと感じ、避けているタスク**は何か？
> それを今週、最初にやったか？」

恐怖は、今取り組むべき最も重要な仕事のシグナル。

## 3. ユーザーから学んだか？

> 「今週、ユーザーと直接話し、**彼らの生活や仕事について新しい発見**があったか？」

ユーザーの目の前にあるものからしか、正しい答えは見つからない。

## 4. 需要の引力を感じたか？

> 「今、私は製品がユーザーから**『引き剥がされる』ような強い需要**を感じているか？」

もしそうでないなら、さらに「スケールしないこと」をしてニーズを深掘りする必要がある。

## 5. 万能ナイフとして動けているか？

> 「私は特定の専門家に留まらず、**あらゆる課題を解決する『万能ナイフ（Super Knife）』**として動けているか？」

創業者は、コーディングだけでなく、営業から掃除まで、勝つために必要なすべてを行う。

---

## 記録テンプレ

```
## 週次自問 YYYY-MM-DD

1. 口コミ: (Y/N) 具体例:
2. 不快タスク: 何を先にやった?
3. ユーザーから学んだこと:
4. 需要の引力: (1-10)
5. 万能ナイフ: 今週やった「コーディング以外」の仕事:
```
