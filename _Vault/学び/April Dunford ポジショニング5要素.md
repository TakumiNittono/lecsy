# April Dunford ポジショニング5要素

> 出典: [[NotebookLM_学習/③ 価格ポジショニング]] / April Dunford

## 原則

ポジショニングは、製品が**特定の顧客層にとって「なぜ世界最高なのか」**を定義する土台。
以下の5要素を**この順番で**検討することが推奨される。

## 5要素（順序付き）

### 1. 代替手段 (Competitive Alternatives)
顧客がこの製品を使わなかった場合、問題をどう解決するか。
**競合他社だけでなく、「Excel」「手書き」「インターンの手作業」**など**現状維持**も含む。

### 2. 独自の能力 (Unique Attributes)
代替手段にはない、自分だけが持つ機能やサービス。

### 3. 価値 (Value)
独自能力が顧客にもたらす利益（時間節約、ミス削減、収益増など）。

### 4. 対象顧客 (Target Market)
その価値を**最も切実に必要とし、高い関心を持つ**顧客の属性。

### 5. 市場カテゴリ (Market Category)
顧客が製品の価値を直感的に理解できるための**「文脈」**。
単なる「管理ツール」ではなく「語学学校向けの成長プラットフォーム」と定義することで、
**適切な期待値を形成**できる。

## lecsy ポジショニング草案

| 要素 | lecsy |
|---|---|
| 代替手段 | 紙の出席簿 / 共有されてない Excel / 汎用 CRM / 気合と根性 |
| 独自能力 | 語学学校特有ワークフロー（講師・生徒・レッスン紐付け）特化 |
| 価値 | 事務時間 50% 削減 + データドリブン生徒ケアによる退学率低下 |
| 対象顧客 | 効率化を**「成長への投資」**と捉える語学学校経営者 |
| カテゴリ | **「語学学校のための OS」「スクール成長加速プラットフォーム」** |

## 1文ポジショニング

> lecsy は、**効率化を成長への投資と捉える語学学校の経営者**のための
> **語学学校向け成長 OS** です。
> Excel や汎用 CRM と違い、**講師・生徒・レッスンの複雑な紐付けに特化**し、
> 事務時間 50% 削減と退学率低下を実現します。
