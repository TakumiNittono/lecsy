# よくある税務ミス5 — 罰金とビジネス破綻の原因

> 出典: [[NotebookLM_学習/④ 米国LLC税務]] / Karlton Dennis, James Baker CPA

## 1. Form 5472 の未提出

- 非居住者 LLC は**所得がなくても提出必須**
- 遅延 = **自動 $25,000 の罰金**
- 対策: [[Form 5472 提出ガイド]]

## 2. 公私混同（Commingling）

- ビジネス用口座で個人の日用品を買う → 資産が混ざる
- **LLC の法的保護（責任限定）が無効化**されるリスク
- 一度「Piercing the Corporate Veil」認定されると、個人資産まで差し押さえ対象
- 対策: ビジネス口座とカードを完全分離。個人費用は一切入れない

## 3. 領収書や記録の不備

- 監査の際、証拠書類がないと**正当な経費であっても控除が否認**される
- PC代、サーバー費用、営業移動費など、全て否認の対象になりうる
- 対策: 全領収書を **Expensify / Receipt Bank / QuickBooks Receipt** 等で即日スキャン。紙は破棄

## 4. 売上税（Sales Tax）の無視

- 物理的な拠点（**ネクサス**）がある州での販売には、所得税とは別に**売上税の徴収・納付義務**
- SaaS も州によって課税対象（Texas, NY, Washington など）
- 対策: 最初の顧客が特定州に偏ったら、その州のルール確認。**TaxJar / Avalara** 等のツール導入

## 5. 不適切な S-Corp の維持

- 給与（Payroll）を適切に支払わない → IRS に否認リスク
- 利益が少ないのに S-Corp のまま → 維持コスト（給与計算代など）で**赤字**
- 対策: [[S-Corp 選択基準]] — MRR $5k 未満は LLC のまま

## lecsy の防衛チェック

- [ ] Form 5472 期限をカレンダー登録（翌年 4/15）
- [ ] ビジネス銀行口座とカードを個人と完全分離
- [ ] Stripe 売上は即 LLC 口座へ、個人口座経由禁止
- [ ] 領収書即日デジタル化のワークフロー確立
- [ ] 顧客獲得時に州を記録（売上税ネクサス監視）
