# SaaS GTM 失敗パターン5

> 出典: [[NotebookLM_学習/⑪ マーケティング・グロース]] / Jason Lemkin (SaaStr)
> MRR $0 から成長段階にある lecsy が避けるべき典型的な失敗

## 1. 製品を実演（Demo）できない営業の採用
創業者からジュニア SDR に至るまで、**デモを完璧にこなせない人物を採用してはいけない**。

→ lecsy は当面、**創業者が全デモを実施**。将来の初採用時もこの基準。

## 2. 「戦略家（Strategist）」の採用
**$10M 以下のフェーズで「戦略家」を自称し、実務（リード獲得やパイプライン構築）を行わない人物は不要**。

→ 初期採用は「泥臭く売れる人」のみ。

## 3. 創業者が早期に営業を離れる
「製品作りに戻りたい」と営業を他者に任せきりにすると、**複雑な商談が成約しなくなる**。
創業者は常に**「最高の営業補佐（Midler）」**であるべき。

→ MRR $30k までは創業者が商談の主役。

## 4. マーケティング予算をゼロにする
成長が鈍化した際にマーケティング（未来）を切り捨て、営業（現在）のみを残すのは間違い。
**小規模でもパイプライン構築の手を緩めてはいけない**。

→ lecsy は MRR $5k までは**広告費ゼロ** + [[Work in Public 戦略]] オーガニック一択でOK。ただし「パイプライン構築」は止めない。

## 5. AI 機能の同等性（AI Feature Parity）の欠如
現代のB2Bにおいて、競合がAIを活用した圧倒的な効率性を謳っている中で、
**自社にAI機能がない場合、コンペで即座に敗北**。

→ lecsy は Deepgram + Claude で既にAI ネイティブ。これを**前面に押し出す**マーケコピーに。

## lecsy セルフチェック（四半期ごと）

- [ ] 創業者が全デモをこなせているか？
- [ ] 新規採用は「戦略家」ではなく「実務家」か？
- [ ] 創業者が営業の最前線にいるか？
- [ ] MRR 停滞時でも「未来への投資（マーケ/コンテンツ）」を継続しているか？
- [ ] AI 機能が競合より明らかに上か？（常にベンチマーク）
