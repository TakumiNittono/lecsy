# SNS 自動化 タイムテーブル

> 親: [[MOC_SNS]] / 決定は [[SNS/自動化_決定ログ]] / 実装は [[SNS/自動化_運用手順]]

## 前提

- **ターゲット**: 日本の indie 起業家 + OPT 日本人エンジニア + 米国挑戦予備軍
- → **全 cron の基準は JST (Asia/Tokyo)**
- あなた (Florida = ET) が寝てる間に投稿される設計
- 週 5-7 投稿 (設計書 P1-P3 配分)

## 日次タイムテーブル

| 時刻 (JST) | ET 換算 | 処理 | 出力 |
|---|---|---|---|
| **04:00** | 15:00 (前日) | **情報収集** `sns-collect.mjs` | `_Vault/SNS/Daily/YYYY-MM-DD.md` |
| **06:00** | 17:00 (前日) | **当日分 3 件生成** `sns-run.mjs --count=3 --dry` | `_Vault/SNS/ドラフト/YYYY-MM-DD.md` + Slack 通知 |
| **08:00** | 19:00 (前日) | **朝投稿** (P1 数字断定 単発) | X 投稿 #1 |
| **12:30** | 23:30 (前日) | **昼投稿** (P3 技術 単発) | X 投稿 #2 |
| **21:30** | 08:30 | **夜投稿** (P2 OPT/米国 スレッド or 単発) | X 投稿 #3 |
| **22:00** | 09:00 | **日次 engagement 収集** `sns-engagement.mjs` | Slack digest |

## 週次タイムテーブル

| 曜日 (JST) | 時刻 | 特別処理 |
|---|---|---|
| **金曜** | 17:00 | **「今週の Lecsy」大玉スレッド** (Build in Public 週次) |
| **日曜** | 20:00 | **週次レビュー digest** (engagement top3 + pillar 配分 + guardrail 頻度) |
| **月初 (1日)** | 09:00 | **月次 KPI レポート** (フォロワー増 / MRR / DM / Lecsy 数字) |

## 時間帯の根拠

### 08:00 JST (朝投稿) — 数字断定単発
- 日本の通勤時間 (7:30-9:00) のピーク
- Florida の user は寝てる → engagement リプに即応できない = 炎上リスク最小
- P1 Build in Public に最適 (「昨日の Deepgram コスト」とか tangible な数字)

### 12:30 JST (昼投稿) — 技術 deep dive 単発
- 日本の昼休み (12:00-13:00)
- OPT/留学生は米国時刻で 23:30 → 寝る前スクロール
- P3 技術系が刺さる (保存されやすい時間帯)

### 21:30 JST (夜投稿) — スレッド or 大玉
- 日本の夜スクロール (20:00-23:00) 後半
- Florida の user は朝 08:30 → 朝コーヒー時の目配り可能
- P2 OPT/米国起業の重いスレッドに最適

### 金曜 17:00 JST (公開デー)
- 週末前の「今週まとめ」が日本の X で伸びやすい
- Work in Public 戦略 の「公開デー」と整合

## エスカレーション (自動停止条件)

以下が発生したら自動で kill switch:

| 条件 | 判定方法 | 動作 |
|---|---|---|
| 3 日連続で guardrail block のみ | sns-engagement.mjs が検知 | `SNS_AUTO_DISABLED=true` + Slack 🚨 |
| 投稿 24h 後の engagement 0 が 3 回 | daily engagement 取得時 | Slack ⚠️ (tone 劣化シグナル) |
| X API 401 / Credit 不足 | 投稿時検知 | 即 Slack 🚨 (自動停止はしない、1発リトライ) |
| OpenAI quota exceeded | 生成時検知 | その日は生成停止、翌日再開 |

## タイムゾーン実装メモ

- Vercel Cron / GitHub Actions の cron 書式は **UTC**
- JST = UTC + 9
- 例: JST 08:00 → `0 23 * * *` (前日 UTC 23:00)
