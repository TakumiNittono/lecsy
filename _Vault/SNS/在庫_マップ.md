## 使い方

- 在庫が 3 本切ったら補充 (P1/P2/P3 各 3 本以上キープ)
- `最終投稿日` 古いものから優先選択 (自動)
- 同じ sourceNote は 30 日間再選ばれない (sns-run.mjs `alreadyPosted` ガード)
- angle を **Q 形式** で書く → gpt-4o が Q+A 構造に即流し込める

## 意図的に除外

- マインド系 (辞めない精神衛生 / Founder Mode / 朝ルーティン) — AI slop と見分けつかない
- 動画素材 (`lecsy-video/content/scripts-en.json`) — 別レイヤー
- Memory 限定ファイル — Vault に本体なし → resolve 不能
- 顔出し系 / personal 系 — memory: [[feedback_voice_not_personal]]
