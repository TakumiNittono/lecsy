## 列の意味

- **id**: `SNS-NNN` (ドラフトファイル内で参照される)
- **ch**: `X` / `LI` (LinkedIn) / `SUB` (Substack)
- **pillar**: `P1` (Build in Public 数字) / `P2` (OPT/米国起業 一次情報) / `P3` (技術 deep dive)
- **sourceNote**: Vault 内のネタ元ファイル (wikilink or \`相対パス\`)
- **angle**: 投稿の切り口 (1行)
- **最終投稿日**: 投稿済み日 (YYYY-MM-DD)

## 使い方

1. **在庫が 3 件を切ったら補充** (枯渇すると動線が止まる)
2. **最終投稿日が古いものから優先** (sns-run.mjs 自動ソート)
3. 投稿後は `最終投稿日` に `YYYY-MM-DD` で埋める
4. 反応が良かったものは **リメイク可**。日付更新すれば再浮上

## 意図的に除外したもの

- **マインド系** (辞めない精神衛生 / Founder Mode / 朝ルーティン 等) — 設計書で禁止
- **動画素材** (`lecsy-video/content/scripts-en.json`) — 別レイヤー、手動管理
- **Memory 限定ファイル** (`[[project_whisperkit_permanent]]` 等) — Vault に本体がないと resolve できない
