## 投稿手順

1. プロフィールの handle / display name / bio を [[SNS/アカウント設計_@takumi_global]] の v2 に合わせて更新
2. 上記 **3 ツイ版** を手動で投稿（自動化を通さない — 1 発勝負）
3. 1/3 ツイをピン留め
4. 完了後、`_Vault/SNS/公開済/YYYY-MM-DD.md` に記録

## なぜ自動投稿じゃなく手動か

- ピン留めは account の顔。guardrail で弾かれた場合に次に進めないと困る
- ユーザーが 1 回目の tone を微調整したい場合がある
- thread の sequence は人間が見て確認すべき
