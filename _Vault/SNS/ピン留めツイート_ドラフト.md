# ピン留めツイート v3 (AI 最新情報発信者)

> 親: [[SNS/アカウント設計_@takumi_global]]
> **v3: AI 業界最新情報を builder 視点で読む account の顔**。Lecsy 言及なし。

## 戦略

プロフィール訪問者に「この account は何をくれるか」を 3 ツイで示す:
- 1/3: 何に答える account か (5 問)
- 2/3: どう書くか (構造 + Voice)
- 3/3: 何を書かないか + CTA

---

## 採用案 (3 ツイ)

### 1/3
```
AI 業界の最新発表を builder 視点で読む account。

毎日答える 5 問:
• 新モデル / API、既存と何が違う？
• AI ツールはどのユースケースに効く？
• API 価格・レート・機能、どう選ぶ？
• 論文・研究、個人開発者が使える部分は？
• 業界の動き、builder はどう読む？
```

### 2/3
```
形式:
Q: [今日のニュース]
A: [数字 1 / 実装判断 / 既存比較]
[taste 1 行]
出典: [URL]

全部 一次ソース直撃、実測比較、煽りぶった切り。
```

### 3/3
```
書かないこと:
• 二次記事の又聞き
• 自分語り / 日報
• 煽り加担 (AGI / revolutionary / 業界初)
• 根拠なき断定

毎週金曜 17:00 JST に「今週の AI 業界メタ」を出します。
DM 歓迎。
```
