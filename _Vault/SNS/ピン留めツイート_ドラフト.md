# ピン留めツイート ドラフト — 自己紹介スレッド

> 親: [[SNS/アカウント設計_@takumi_global]]
> 用途: @takumi_global のプロフィールに pin する **大玉スレッド**。
> これが訪問者の最初の判断材料になる。

## 戦略

プロフィール訪問者の 8 割はこれを読む。読了後に「フォローする価値あるか」が決まる。

3 つの責任を果たす:
1. **誰か** を伝える (OPT 日本人、Florida、個人開発者)
2. **何を作ってるか** を伝える (Lecsy = 米国 B2B 講義AI)
3. **何を発信するか** を約束する (Build in Public / 週次進捗)

## 推奨

**A 案 (5 ツイ)** でいく。

理由:
- 訪問者の info hunger を 1 スレッドで満たせる (4/5 が pillar 宣言、5/5 が約束)
- 各ツイを単独で引用 RT されやすい (2/5 の "この市場が存在しない国で〜" は刺さる)
- pin に置く資産として質量がある

## 投稿手順

1. handle を `@takumi_global` に変更 (まだなら)
2. Display name を `Takumi Nittono / Lecsy` に
3. Bio を [[SNS/アカウント設計_@takumi_global]] の 120 字版に
4. Header image を設定 (案1 テキスト or 案2 製品スクショ)
5. **A 案を手動で投稿** (自動化を使わない — これは 1 発勝負)
6. 投稿後、1/5 ツイをピン留め
7. 完了後、2本目以降から `sns-run.mjs` が回す
