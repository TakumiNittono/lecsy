# 02. 競合分析

## 直接競合

### 🟥 Otter.ai
- 価格: Free(限定)/ Pro $16.99/月
- ポジション: ミーティング文字起こし(英語中心)
- 強み: ブランド認知、Zoom/Meet 統合、要約品質
- **弱み(Lecsy が突く点)**:
  - クラウド必須 → 電波悪い講義室で機能停止
  - 集団訴訟中(2025-):学生データの AI 学習利用問題
  - 英語中心、多言語要約が弱い
  - 無料枠が極小(月600分)
- Lecsy の刺し方: **"Otter より無料、オフライン、プライバシー"**

### 🟥 Notta
- 価格: Free(120分/月)/ Pro $13.99/月
- ポジション: 多言語議事録(58言語)
- 強み: 言語数、Web UI
- 弱み: 完全クラウド、学生向けでない、要約は英/中ローカライズ弱
- Lecsy の刺し方: **"Notta よりオフライン、学生特化、試験モード"**

### 🟧 PLAUD NOTE(ハードウェア)
- 価格: デバイス $159 + サブスク
- ポジション: AI ボイスレコーダー物理デバイス
- 弱み: 買い切り高額、物理デバイス必須
- Lecsy の刺し方: **"iPhone だけで PLAUD と同じ事が無料で"**

### 🟨 Goodnotes 6(Audio Note 機能)
- 強み: ノート大手の追加機能、学生利用多
- 弱み: AI 要約弱、文字起こし後発
- Lecsy の刺し方: **文字起こし品質 + 多言語要約で勝負**

### 🟨 Voice Memos(Apple 純正)
- 強み: プリインストール、無料
- 弱み: 要約・試験モードなし、検索性ゼロ
- Lecsy の刺し方: **"録音した後"の学習体験**

### 🟨 Notability / Apple Notes
- 録音機能あり。文字起こしは弱い、AI要約ない。

## App Store 検索面の競合マップ

| 検索語 | 現在の1位帯 | Lecsy の参入余地 |
|---|---|---|
| lecture recorder | Lecture Recorder SE 等古参 | 高(UI が古い) |
| transcribe audio | Otter, Transcribe | 中(offline 訴求) |
| ai note taking | Otter, Notta, Fathom | 中 |
| 講義 録音 | 古参録音アプリ | **高(AI+無料で勝てる)** |
| 文字起こし | Notta, CLOVA Note | 中 |
| otter alternative | ほぼ空白 | **超高** |
| offline transcription | 空白気味 | **超高** |
| 留学生 ノート | 空白 | **超高(B2Bにも繋がる)** |

## ポジショニング・ステートメント(覚える)

> **Lecsy は、留学生と学生のための"オフラインで動く Otter"。**
> クラウドに音声を送らず、無料で無制限に録音・文字起こしし、
> 9言語で AI 要約して試験対策まで行う唯一の講義アプリ。
