## 弱点(ASO 観点で今すぐ潰す)
| 弱点 | 影響 | 対応 |
|---|---|---|
| アプリ名が "lecsy" だけ | 検索露出がブランド名依存 | → `lecsy: Lecture Recorder AI` 形式に(+14文字のサブキーワード枠を獲得) |
| サブタイトルがコピー的で検索語不足 | サブタイトル30文字は**ランキング対象**なのに"×"記号で無駄遣い | → キーワード濃縮型に書き換え(04参照) |
| キーワードフィールドに重複・弱語 | "勉強""復習"はボリューム低 | → "オトター""AI要約""議事録""留学"等に置換 |
| 英語キーワードに "otter alternative" "offline" の直接狙いなし | 競合指名検索を逃している | → 英語 keywords を改訂 |
| ローカライズが JP/EN のみ | ES/FR/DE/PT/IT/RU/HI は**文字起こし対応済**なのにストアロケール未展開 (KO/ZH は非対応のため除外) | → 06 参照、順次展開 |
| スクショに "無料"・"オフライン"・"音声は端末から出ない" の**バッジ**なし | 差別化が伝わらない | → 05 参照 |
| レビュー依頼タイミングが無い(推定) | ★獲得が偶発的 | → 07 参照 |
| App Preview 動画なし(推定) | CVR 最大20% の機会損失 | → 05 参照 |
