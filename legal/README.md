# Lecsy Legal & Compliance Package

> ⚠️ **DISCLAIMER:** これらの文書はテンプレートです。実際の契約締結前に必ず適切な法域の弁護士のレビューを受けてください。Lecsy は法的助言を提供するものではありません。

## ファイル一覧

| ファイル | 用途 | レビュー要 |
|---|---|---|
| `MSA.md` | Master Services Agreement（基本契約） | 米国弁護士 |
| `DPA.md` | Data Processing Agreement（GDPR + UK GDPR） | EU 弁護士 |
| `BAA.md` | Business Associate Agreement（HIPAA、希望校のみ） | 米国弁護士 |
| `SLA.md` | Service Level Agreement | 自社作成可 |
| `AUP.md` | Acceptable Use Policy | 自社作成可 |
| `SUBPROCESSORS.md` | サブプロセッサ一覧 | 自社更新 |
| `SECURITY_OVERVIEW.md` | 営業同行用 1 ページャ | 自社作成可 |
| `SOC2_Roadmap.md` | SOC2 取得計画 | 自社作成可 |
| `RFP_TEMPLATE.md` | RFP 頻出 30 項目への即答 | 自社作成可 |
| `INSURANCE.md` | サイバー保険の付保証明 | 保険会社発行 |
| `CAIQ_Lite_Lecsy.md` | CSA CAIQ-Lite 回答 | 自社作成可 |

## 使い方

1. 営業初回コンタクト時: `SECURITY_OVERVIEW.md` を PDF 化して送付
2. パイロット開始時: `MSA.md` + `SLA.md` を提示
3. EU/UK 顧客: `DPA.md` を追加
4. 大学/エンタープライズ: `CAIQ_Lite_Lecsy.md` + `SOC2_Roadmap.md` を提供
5. RFP 受領時: `RFP_TEMPLATE.md` から回答を組み立てる
