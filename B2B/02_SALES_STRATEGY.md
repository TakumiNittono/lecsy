## 1. プライシング

### 語学学校向け（Phase 1-2）

| プラン | 価格 | 対象 | 含まれるもの |
|--------|------|------|------------|
| **Teacher Free** | $0 | 先生個人 | 無制限文字起こし。まず先生に体験させる |
| **Classroom** | $500-$1,000/学期 | 1クラス ~20名 | 全生徒にPro機能、利用レポート |
| **School** | $2,000-$5,000/年 | 学校全体 ~100名 | 無制限席、管理画面、レポート |
| **Chain** | $5,000-$25,000/年 | 複数キャンパス | 全拠点対応、専任サポート |
| **Corporate** | $5,000-$15,000/年 | 法人語学研修 | カスタムレポート、請求書払い |

**ポイント:**
- 先生への無料提供が入口 → 生徒に広がる → 学校契約
- 学期単位の価格設定（語学学校は学期ごとに生徒が入れ替わる）
- 言語ラボ($5,000-$50,000/教室)の10分の1以下をアピール

### 大学向け（Phase 3-4）

| プラン | 価格/年 | 対象 |

| Department Pro | $3,000-$8,000 | ~200席 |
| Campus-Wide | $15,000-$50,000 | 無制限、SSO |

## 3. メール営業の実行計画

### ボリューム & ペース

| 指標 | 目標 |
|------|------|
| 1日のメール送信数 | 15-25通（パーソナライズ必須） |
| 1件のメールシーケンス | 4-7通 / 2-4週間 |
| メール間隔 | 2-3日 |
| 期待返信率 | 10-20%（語学学校は大学より高い） |
| 期待ミーティング率 | 5-10% |
| 25通/日 → 週のミーティング | 5-10件 |

### 担当者の見つけ方

- 学校ウェブサイトのスタッフページ（ほぼ全校で公開）
- LinkedIn（「Language School Director Florida」で検索）
- CEA/ACCET認定校リスト → ウェブサイトへ
- ESLDirectory.com → 学校検索
- Google Maps → "[City] English school" で検索
- Apollo.io / Hunter.io（メールアドレス検索、無料枠あり）

### パーソナライズのコツ

- 学校名、担当者名を必ず入れる
- 「御校のウェブサイトで[プログラム名]を拝見しました」
- 学校の国籍構成に言及（「70カ国の生徒がいらっしゃるということで、12言語対応が活きます」）
- 近くにいることを強調（「Ocalaから車で[X]分なので、いつでも伺えます」）

## 6. パイロットプログラム

### 構成

| 項目 | 内容 |
|------|------|
| 期間 | 1学期（または最低4-8週間） |
| 範囲 | 1-3クラス、20-50人の生徒 |
| 費用 | **無料** |
| Lecsy提供内容 | 全参加者にPro機能、専用サポート、週次チェックイン |
| 学校に依頼すること | フィードバック、利用データ使用許可、成功時のテスティモニアル |
| 成功指標（事前定義） | 利用率、リスニングテストスコアの変化、生徒満足度 |
| パイロット後 | 結果レポート提出 → 有料プラン提案 |

### なぜ無料パイロットが必要か

- 語学学校は新しいツールに慎重。無料なら試しやすい
- 使い始めると先生が依存する → 更新率が高い
- 実データが次の学校への営業資料になる
- $0なので調達プロセス不要 → 即開始できる

## 7. チェーン攻略戦略

1校で成功 → チェーン全体に展開する最も効率的な方法:

### ステップ

```
1. 最寄りのキャンパスでパイロット実施
2. 成功データ（利用率、生徒フィードバック）を収集
3. キャンパスの先生/ディレクターから推薦を獲得
4. HQ（本部）のオーナー/CEOにアプローチ
5. 「[キャンパス名]での結果です。全キャンパスに展開しませんか？」
6. チェーン全体契約を締結
```

## 8. 語学学校 → 大学へのステップアップ

```
語学学校での実績
  │
  ├→ ケーススタディ: 「リスニングスコアがX%向上」
  ├→ 先生のテスティモニアル
  ├→ 利用データ: 「X人の生徒がY時間使用」
  │
  ↓
大学附属IEPに営業（語学学校と同じピッチが使える）
  │
  ├→ UF ELI（Gainesville、35分）
  ├→ UCF IEP（Orlando）
  ├→ INTO USF（Tampa）
  │
  ↓
IEP経由で大学本体のDisability Servicesに紹介
  │
  ├→ ADA Title IIコンプライアンス営業に切り替え
  ├→ 高単価契約（$15,000-$50,000/年）
  │
  ↓
キャンパスワイド契約
```
