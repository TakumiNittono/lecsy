## 最重要ポイント

1. **語学学校から始める** — 意思決定が速い（2-6週間）、12言語がドンピシャ、チェーン展開で一気にスケール
2. **唯一無二の差別化** — 12言語リアルタイム文字起こし + オンデバイスAI + オフライン動作。この3つを同時に持つ競合はゼロ
3. **12ヶ月が勝負** — 対面（学校訪問・デモ・信頼構築）に全集中。OPT後もリモートで回る仕組みを同時に作る
