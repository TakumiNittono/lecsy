# 08. OPT終了後の事業継続計画
**OPT: 2026年6月1日〜2027年5月31日（12ヶ月のみ、延長なし）**

---

## 大前提

12ヶ月間で「人がいなくても回る仕組み」を作り切る。
語学学校の契約は年間自動更新。リモートでも営業・サポート可能。

## ビザオプション

### Option 1: 帰国 + リモート運営（最も現実的）

| 項目 | 詳細 |
|------|------|
| 実現性 | ★★★★★ |
| 概要 | 日本/他国に帰国、LLCはフロリダに残してリモート運営 |
| メリット | ビザ問題なし、生活費安い、LLCのまま米国顧客対応可 |
| デメリット | 対面営業不可、時差（日本-FL: 13-14時間） |
| 準備 | Registered Agent契約、リモート営業体制確立、自動化ツール |
| 営業 | 新規はZoom中心、既存はメール/Slackで継続 |

## リスクと対策

| リスク | 対策 |
|--------|------|
| リモート移行で顧客離れ | 年間契約推進、自動更新設定、3月にリモーテスト実施 |
| 12ヶ月で十分な売上が出ない | 語学学校に集中（契約が速い）、コミュニティカレッジも並行 |
| 新規営業がリモートで難しい | 紹介ベース営業、パートタイムセールス採用、Zoomデモ |
| 競合追い上げ | LMS連携・年間契約で切替コストを高める |
