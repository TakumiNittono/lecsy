## 4. Lecsyが語学学校に刺さる理由

### 4A. ペインポイント（語学学校が抱える課題）

1. **リスニングの壁**: 生徒はテキストは読めるが、自然なスピードの英語が聞き取れない。既存教材はスクリプト付きの録音のみ
2. **リアルタイム文字起こしがない**: 先生が話す→生徒は聞く→聞き取れなかった部分は消える
3. **言語ラボが高額**: SmartClass/Robotel = 1教室$5,000-$50,000+。全教室に設置不可
4. **クラウドツールのプライバシー懸念**: Otter.ai等は生徒データをクラウドに送信
5. **WiFi問題**: 教室のWiFiが不安定でクラウドツールが使えない
6. **多言語サポートが弱い**: 多くのツールは英語のみ。アラビア語・ヒンディー語・韓国語等の母語に対応していない
7. **生徒のコスト負担**: ESL学生は低所得層も多い。Otter Pro $16.99/月は高すぎる

### 4B. Lecsyが解決すること

| ペインポイント | Lecsyの解決策 |
|-------------|-------------|
| リスニングの壁 | リアルタイム文字起こしで「聞き逃し」を可視化 |
| 復習できない | 録音+テキストで何度でも復習可能 |
| 言語ラボが高い | 生徒のiPhoneが言語ラボになる。$2,000-$5,000/年で学校全体 |
| プライバシー | 100%オンデバイス。データはクラウドに出ない |
| WiFi不安定 | オフラインで完全動作 |
| 多言語非対応 | 12言語対応で全生徒の母語をカバー |
| 生徒のコスト | 個人利用は無料 |

### 5C. Lecsyのブルーオーシャン

**「12言語リアルタイム文字起こし」+「オンデバイスAI」+「低価格」を同時に持つ競合はゼロ。**

- Otter.ai → 英語のみ、クラウド依存
- SmartClass → 高額ハードウェア必要
- ELSA Speak → 発音のみ、文字起こしなし
- Jamworks → クラウド依存、$18.99/月

### 5D. 営業中の競合対応トーク

**「Otter.aiと何が違う？」**
> Otterは英語のみでビジネス向けです。Lecsyは12言語対応で語学学習に特化しています。最大の違いは全てがデバイス上で処理されること。生徒のデータはクラウドに送られませんし、WiFiがなくても動きます。

**「うちは言語ラボ（SmartClass等）を使っている」**
> 言語ラボは素晴らしいですが、1教室$5,000-$50,000以上かかりますよね。Lecsyなら年間$2,000-$5,000で学校全体をカバーできます。生徒のiPhoneがそのまま言語ラボになります。既存の言語ラボとの併用も可能です。

**「生徒はDuolingoを使っている」**
> Duolingoはゲーミフィケーションによるドリル学習です。Lecsyは実際の授業・会話をリアルタイムで文字起こしする全く別のツールです。先生の話す「本物の英語」を理解する力を鍛えます。補完関係にあります。
