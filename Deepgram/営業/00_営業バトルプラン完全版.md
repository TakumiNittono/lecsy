## 2. メール戦略 — 3層アプローチ

全ターゲットを3つの層に分けて、それぞれ違うアプローチで攻める。

### 層1: 個別校ファイル11校（最重要、全てカスタムメール）

| 学校 | チャネル | 理由 |
|------|---------|------|
| Liberty Language Academy | WhatsApp + LinkedIn | WhatsApp公開、ブラジル人CEO |
| Santa Fe College ESL | 電話 + メール | 地元、直接訪問可 |
| College of Central Florida | 電話 + メール | 車30分圏 |
| Celebration Language Institute | 電話 + メール | Orlando日帰り |
| International Preparatory School | フォーム + メール | F-1特化校 |
| Tampa Language Center | フォーム + 電話 | 2拠点、価格訴求 |
| FELI (Dunedin) | メール + LinkedIn | CEA認定 |
| Lingua Language Center | メール + LinkedIn | Miami Brickell |
| CCLS Miami | メール + LinkedIn | 30年老舗 |
| TLA Fort Lauderdale | メール + LinkedIn | 新施設投資中 |
| English Language Academy Orlando | メール + LinkedIn | 受賞校 |

**原則**: 1校あたり最低3チャネルで同時に攻める。WhatsApp > 電話 > LinkedIn > メール > フォーム。

### 層2: CSVリード427人（Tier1-3を優先、パーソナライズメール）

CSVの構成:
- **Tier1（Director級）**: 最優先。1人ずつ名前・学校・役職でカスタムメール
- **Tier2（Coordinator級）**: Tier1が無反応時の迂回ルート
- **Tier3（DRC Director）**: 別角度（ADA/アクセシビリティ）で攻める
- **Tier4（Faculty）**: 今は後回し。パイロット校が決まったら内部推薦者として使う

### 層3: LinkedIn DM営業（並行して毎週10件）

LinkedInは即売り込みNG。ウォームアップ→接続→価値提供→DM の4ステップ。

## 3. 全校カスタムメール集

### 送信ルール
- **件名**: 学校名を入れる。汎用的な件名は開封されない
- **本文**: 3段構成（共感フック → lecsyの1行説明 → CTA）
- **長さ**: 150語以内。それ以上は読まれない
- **添付**: 1-pager PDF（2通目以降。初回は添付なし＝スパムフィルター回避）
- **送信時間**: 火曜〜木曜の午前9:00-10:30（開封率最高帯）
- **送信元**: takumi@lecsy.app（独自ドメイン必須。Gmail個人アドレスは信用されない）

<a id="reply-playbook"></a>
## 5. 返信が来たらどうするか — レスポンスプレイブック

### 返信の種類と対応

| 返信パターン | 対応 | 目標 |
|-------------|------|------|
| 「面白そう、話聞きたい」 | **30分以内**に返信。Zoom or 訪問日程を2択で提案 | 48時間以内にデモ日確定 |
| 「資料送って」 | 1-pager PDF + 60秒デモ動画を即送信 → 3日後にフォロー | 1週間以内にデモ日確定 |
| 「今は忙しい」 | 「いつなら？」と聞く。具体日程なければ2週間後にリマインド | カレンダーに再コンタクト日を入れる |
| 「予算ない」 | 「**4週間完全無料パイロット**です。請求ゼロ」 | 無料であることを再強調 |

## 11. マインドセット — 全員に送る覚悟の実践

### 数字で考える

```
層1（個別校）: 11校 × 3チャネル = 33コンタクト
層2（CSV Tier1-3）: ~140人 × メール = 140コンタクト
合計: 約170コンタクト

返信率15%として = 25-26件の返信
→ デモ設定率50% = 12-13件のデモ
→ パイロット率40% = 5件のパイロット
→ 有料化率60% = 3件の有料契約（Year 1）

つまり: 170通送れば3契約。1通あたりの価値 = 約$60（ACV $3K × 3 ÷ 170）
```

### 拒否されても大丈夫な理由

- 170通中155通は無視 or 拒否される。**それが正常**
- 「No」は情報。「Why no?」を聞けば次に活かせる
- 1校目の契約が取れた瞬間、2校目以降は**10倍楽になる**（ケーススタディの力）

### 毎朝のルーティン

```
8:00 - Google Sheet開く → 今日のフォロー対象を確認
8:15 - フォローメール/電話を先にやる（新規より優先）
9:00 - 新規アウトリーチ（メール or WhatsApp）
10:00 - LinkedIn活動（コメント + 接続申請）
10:30 - プロダクト改善 or デモ準備
```

## 14. 失敗パターンと対策

| やりがちなミス | なぜダメか | 対策 |
|-------------|----------|------|
| メール送ってから返信を待つだけ | 返信率15%。待ってる間に1ヶ月経つ | 3日後フォロー必須。別チャネルでも攻める |
| 完璧なメールを書こうとして1通30分かかる | 427人に送れない | テンプレ + パーソナライズ1行、1通5分以内 |
| 断られて落ち込む | 155通は無視される。それが営業 | 数字で考える。170通で3契約 |
| プロダクト改善に逃げる | 営業が怖いから開発に戻る | 午前は営業、午後にプロダクト。順番を守る |
| 全校同じメールを送る | 「コピペ」がバレて開封すらされない | 学校名 + 固有事実1行 + 担当者名。最低限のパーソナライズ |
| アポなしで訪問して門前払い | 準備不足の印象 | アポなし訪問は「資料を渡すだけ」。デモはアポ取ってから |
| デモでしゃべりすぎる | 相手が話す時間がないと購買意欲が下がる | 15分中、相手に5分しゃべってもらう。質問を聞く |
| パイロット開始後に放置 | 使われないまま終わる → 有料化しない | 週1で学校に顔出し or メール。利用データを共有 |
