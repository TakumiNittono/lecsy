# LinkedIn攻略：大学・カレッジ ESL/IEP プログラム

> **目的**: 大学付属の英語学習プログラム（ESL / IEP / ELI）にLinkedIn経由でlecsyを売り込むための「誰と繋がるか」完全マップ。
> **前提**: 既存の[`_営業プレイブック共通.md`](./_営業プレイブック共通.md)のLinkedIn DMテンプレを使う。本ファイルは**ターゲット選定編**。

---

## 🎯 大学ESLが小規模語学学校と違うポイント

| 観点 | 私立語学学校 | 大学・カレッジESL/IEP |
|---|---|---|
| 決裁者 | オーナー1人 | Director + 副学長 + 予算委員会 |
| 予算サイクル | 即決可能 | **年度予算（春に翌年度確定）** |
| 売れる角度 | コスト削減 | アクセシビリティ + 国際学生満足度 + コンプライアンス |
| 営業期間 | 2〜4週 | **3〜9ヶ月** |
| 1契約規模 | $299〜599/月 | **$1,000〜5,000/月**（複数学部展開可） |
| 紹介の力 | 弱い | **強い**（業界が狭い、紹介で次々広がる） |

→ 戦略：**短期は私立語学学校、中長期は大学ESL**。両方並走。

## 👥 ターゲット役職マップ（5階層）

### Tier 0：究極の決裁者（直接攻めない、知っておく）

| 役職 | 攻め方 |
|---|---|
| Provost / Vice Provost for International Affairs | 直接DMはNG。下から登った後、Director経由で会う |
| Vice President for International Programs | 同上 |
| Dean of International Education | 同上 |
| Vice Chancellor for International Education | 同上 |

**理由**：上層部はDM大量に来てて全部無視する。Director経由で「うちのESLが導入したいAI、投資効果あるから紹介します」と上に上げてもらうのが王道。

---

### Tier 1：本命（LinkedIn最優先で繋がる）

**実質的に予算と導入を決める層。ここが本丸。**

| 役職タイトル例 | 検索キーワード |
|---|---|
| Director, English Language Institute (ELI) | `"ELI Director"` `"English Language Institute"` |
| Director, Intensive English Program (IEP) | `"IEP Director"` `"Intensive English Program Director"` |
| Director, ESL Program | `"ESL Director"` `"Director of ESL"` |
| Director, Academic English Program | `"Academic English Director"` |
| Director, English Language Programs | `"English Language Programs"` |
| Executive Director, ELI/IEP | `"Executive Director" + "English"` |
| Director, International Language Center | `"International Language Center"` |

**この人たちに必ず会う**。1校1人。

#### 特徴・響くポイント
- 多くは**TESOL博士・修士**（教育系アカデミック）
- KPIは「学生満足度」「学業成績」「再登録率」「アクセシビリティ準拠」
- **コスト削減はあまり響かない**（公立だと予算は使い切り型）
- 響くのは：「**国際学生が授業についていけない問題**を解決する」「**ADA/Section 508準拠**のアクセシビリティを強化」「**FERPA-safe**で監査リスクなし」

#### LinkedIn DMの一行目（彼ら専用）
```
Hi [Name], I noticed [School]'s IEP serves students from [main countries].
I built lecsy to solve the exact gap I faced as a Japanese student in
US classrooms — and I'd love your honest feedback as someone who lives
this problem daily.
```

→ 「売り込み」じゃなく「**業界専門家としての意見をください**」のスタンスが効く。

---

### Tier 2：副官・推薦者（Tier 1への橋渡し）

| 役職 | 役割 |
|---|---|
| Assistant Director, IEP/ELI | Director不在時の意思決定、新ツール評価係 |
| Associate Director, English Language Programs | 同上 |
| Academic Coordinator, ESL | カリキュラム編成、現場との橋渡し |
| Curriculum Coordinator, ESL/IEP | 教材選定権あり |
| Program Coordinator, English Language | 日々のオペ、学生との接点最多 |

**Tier 1がDM返さない時の迂回ルート**。Coordinatorは比較的DMに返信する。「Director直接より話聞いてくれた」というケース多い。

#### 検索例
- `"ESL Coordinator" + Florida`
- `"IEP Coordinator"`
- `"Curriculum Coordinator" + ESL`

---

### Tier 3：横の決裁者（並走攻略で勝率2倍）

**ここが大学攻略の隠し玉。ESL予算じゃなくアクセシビリティ予算で買える。**

| 役職 | なぜ重要 |
|---|---|
| **Director, Disability Resource Center (DRC)** | ★★★ 文字起こしツールの最大ユーザー。ADA準拠で**買わざるを得ない予算枠**を持つ |
| **Director, Student Accessibility Services** | DRCの別名。同じく重要 |
| Coordinator, Accommodations & Captioning | 文字起こし・キャプション専門。ニーズ100%一致 |
| Director, Center for Teaching & Learning (CTL) | 教員向けツール導入の窓口。ESL教員もここ経由で導入可 |
| Director, Educational Technology | EdTech全般の評価・推奨を出す部署 |
| Instructional Designer (Language Learning) | 個別講座のツール選定者 |
| Director, Academic Support Services | チューター・学習支援の統括 |

#### なぜDRCが重要か
- ESL Directorは「いい話だけど予算ない」と言いがち
- DRC Directorは「**ADA訴訟リスクがある**」と言うとすぐ動く
- DRCの予算は連邦法で守られてる枠 → 削れない
- DRC経由で導入 → 全学に水平展開しやすい

#### DRC向けDM
```
Hi [Name], I built lecsy — an on-device AI transcription tool that
runs offline on student iPhones. We're FERPA + Section 508 compliant
out of the box. I'd love 15 minutes to show how it can supplement your
captioning workflow without adding to your team's manual workload.
```

→ ESL向けと違い、**コンプライアンス + 業務軽減**の角度で話す。

#### 検索例
- `"Disability Resource" + Director + Florida`
- `"Accessibility Services" + Director + university`
- `"Captioning Coordinator"`
- `"Educational Technology" + Director + community college`

---

### Tier 4：現場（ユーザーと内部推薦者）

| 役職 | 攻め方 |
|---|---|
| ESL Instructor / Faculty | 「現場の声」として導入を後押ししてくれる |
| Lecturer, English as a Second Language | 同上 |
| TESOL Faculty (PhD) | 業界権威。1人押さえると紹介で5校広がる |
| Adjunct ESL Faculty | 複数校掛け持ち→ **1人で複数校に展開可** |
| Language Lab Manager / Coordinator | 既存ラボとの連携を握る人 |
| Lead Instructor, Academic English | 教員チーム内の影響力者 |

**狙い方**：DMは送らず、**LinkedIn投稿にコメント**から始める。「現場の声をシェアしてくれてありがとう」系。3〜4回やり取りした後にfollow→connect→DM。

---

### Tier 5：国際学生支援側（別ルート、紹介源）

ESLに直接関係ないが、国際学生という共通顧客を持つので**紹介経路として強い**。

| 役職 |
|---|
| Director, International Student Services (ISS) |
| International Student Advisor |
| Director, Office of International Affairs |
| Coordinator, F-1/J-1 Student Services |
| Cross-cultural Programs Coordinator |
| International Student Recruitment Director |

#### 響くメッセージ
「ESLに通えない・通うのは嫌な国際学生でも、**正規授業の理解が上がる**ツールです。あなたの担当学生のリテンションに直結します。」

## 📋 アプローチ手順（接続〜デモまで）

### Step 1：プロフィール側を整える（送る前に）
- [ ] ヘッドライン: `Founder, lecsy | On-device AI transcription for ESL/IEP programs | Ex-international student, now helping FL universities`
- [ ] About: 200字で「日本人留学生として講義についていけなかった経験 → lecsyを作った → FL大学ESLに4週無料パイロット提供中」のストーリー
- [ ] Featured: lecsyデモ動画60秒 + LP リンク
- [ ] Banner: lecsyのスクショ + "FERPA · Section 508 · Offline · 12 Languages"
- [ ] Activity: 直近2週間でTESOL/ESL系投稿に**5件以上コメント**して「業界の人」感を出す

### Step 2：ウォームアップ（5〜7日）
- ターゲットの直近の投稿に**いいね + コメント**（中身のある1〜2文）
- アルゴリズムが「相互の人」と認識し始める
- 相手のフィードに自分の投稿が出やすくなる

### Step 3：接続申請（パーソナルメッセージ付き）
```
Hi [Name], I lead lecsy, an on-device AI transcription tool for ESL
classrooms. I've been following your posts on [specific topic from
their profile/post] and would love to connect with someone shaping
this space at [School].
```
→ **売り込み禁止**。共通点・敬意のみ。

### Step 4：承認後3日待つ → 価値提供メッセージ（DMじゃなくフィード経由が良いことも）
```
Hi [Name], thanks for connecting. I just published [a short post / a
1-pager] on the 3 things I learned doing pilots with FL ESL programs.
No ask — happy to share if useful.
```

### Step 5：1週間後、本DM（プレイブックのテンプレB）
- 既存の[`_営業プレイブック共通.md`](./_営業プレイブック共通.md) のLinkedIn DMテンプレ
- **必ず学校固有の事実**を1つ入れる（学生数、出身国TOP3、最近のニュース）

### Step 6：返信→15分Zoom→Tier 3も巻き込んだ対面デモ提案

## ⚠️ やってはいけないこと

- ❌ **接続即DM**で売り込む（Spam扱い、アカウント制限の元）
- ❌ **同じテンプレを誰にでも送る**（「コピペした感」が即バレる）
- ❌ **フォローアップを4回以上**送る（嫌われる、3回まで）
- ❌ **LinkedInで価格・契約条件を長文で送る**（電話/Zoomに誘導）
- ❌ **「日本人だから〜」を売りにしすぎる**（最初の1回はOK、2回目以降は鼻につく）
- ❌ **接続申請を20件/週超え**でLinkedIn警告
- ❌ **業界用語を間違える**（IEP/ELI/ESLの違いを把握してから送る）

## 📌 業界用語ミニ辞書

| 略語 | 正式名 | 意味 |
|---|---|---|
| ESL | English as a Second Language | 第二言語としての英語（一般用語） |
| ESOL | English for Speakers of Other Languages | ESLの別名、K-12でよく使う |
| EFL | English as a Foreign Language | 外国語としての英語（米国外で使う） |
| EAP | English for Academic Purposes | 学術目的の英語 |
| ESP | English for Specific Purposes | 専門目的（医療英語など） |
| IEP | Intensive English Program | 集中英語プログラム（大学付属） |
| ELI | English Language Institute | 英語学習機関（多くは大学付属） |
| TESOL | Teaching English to Speakers of Other Languages | 業界最大の専門団体 / 学位名 |
| TEFL/TESL | Teaching English as a Foreign/Second Language | 教員資格 |
| AAIEP | American Association of Intensive English Programs | IEP業界団体 |
| UCIEP | University & College IEP | 厳格な認定団体 |
| NAFSA | Association of International Educators | 国際教育最大手団体 |
| ADA | Americans with Disabilities Act | 障害者法（DRC関連） |
| FERPA | Family Educational Rights and Privacy Act | 学生データ保護法 |
