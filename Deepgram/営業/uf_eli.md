## 🎯 なぜここが最優先か

### 地の利
- **たくみがGainesville在住**（徒歩/自転車でキャンパスアクセス）
- 何度でも顔出せる、対面デモの往復コストゼロ
- UF ELI は UF キャンパス内 Norman Hall 付近

### 事業上の価値
- **UF flagship事例化**: "Used at University of Florida" は他IEPへの営業で絶大
- **予算独立**: UF本体の HECVAT 調達プロセスを（部分的に）回避できる可能性
- **多言語学生層**: Deepgram Multilingual の価値が直接刺さる
- **日本人学生もいる**: たくみのストーリーが直接応援される

### 戦略的タイミング
- **4-6月**: 春セッション終盤 → Directorが暇、デモ受け入れやすい
- **Fall 2026（8月）開始セッション**でパイロット目標
- E-2ビザ経由でUF進学計画もあるため、**将来の自分の在籍先**でもある（長期関係）

## 🎯 アプローチ戦略（3段階）

### Phase 1: Discovery（Week 1-2）
1. UF ELI公式サイトで現役Director/Coordinatorを特定
2. LinkedInで本人検索、プロフィール確認
3. UF ELI学期カレンダー確認（次セッション開始日）
4. 公式フォーム or info@ でアポ打診メール送信

### Phase 2: First Contact（Week 2-3）
1. 15分 discovery call（Zoom or 対面）
   - 「現状のESL学生の英語理解困難をどう支援してますか？」聞く
   - Accessibility予算の有無を確認
   - lecsyはまだ売らない、関係構築のみ
2. 7日以内にキャンパス訪問（コーヒー持参）

### Phase 3: Demo & Pilot Proposal（Week 3-4）
1. 15分デモ（Accessibilityシーケンス入り）
2. 無料4週パイロット提案
3. Fall 2026セッションでの本格導入を視野

## 🎬 デモシーケンス（UF ELI向けカスタム）

| 時間 | 内容 | 目的 |
|------|------|------|
| 0:00-1:30 | 自己紹介：日本人・Gainesville在住・未来UF学生 | 共感・長期関係のシグナル |
| 1:30-4:00 | サンプル英語講義（TEDx学術系）をiPhoneでライブ録音 | 基本機能 |
| 4:00-6:00 | 言語切替：中国語ナレーション → Multilingualで英中混在を認識 | 多言語学生層に刺す |
| 6:00-8:00 | **「音声どこ行った？」の問い**：Deepgramダッシュボード見せて「処理後即破棄」証明 | FERPA訴求 |
| 8:00-10:00 | VoiceOver ON → アプリ操作→ Accessibility訴求 | Disability Services絡める |
| 10:00-13:00 | B2Bダッシュボード：クラス別利用統計・学生別進捗 | 経営・運用価値 |
| 13:00-15:00 | 価格（Starter $349/月）+ 無料パイロット条件 + Q&A | クロージング |

**必殺技**: Deepgramダッシュボードを見せて「音声は1秒も残らない」を目視で証明。FERPA訴求が"言うだけ"から"見える"に変わる。

## 💳 購買パス仮説

### Option A: ELI Director P-card（最速）
- $349/月 は P-card 閾値内
- Director裁量で購入可能性あり（UF ELIの調達ルール要確認）
- Procurement回避

### Option B: UF本体 Procurement 経由
- HECVAT + DPA + UF Data Security Addendum 必要
- 3-12ヶ月
- $5,000超契約は正式プロセス

### Option C: パイロット無料→次年度予算化
- 4週パイロットは契約不要
- 成功体験を元にFY27予算に組み込み要請
- Spring 2027 or Fall 2027 有料化

**推奨**: C → A → B の順で提案。まずは無料パイロット、成功したらP-card月額、年間契約はProcurement経由。
