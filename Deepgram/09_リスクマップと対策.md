## 🚨 最優先リスク 詳細対策

### リスク 1: 6/1 ローンチ遅延

**シナリオ**: Deepgram統合・Bilingual実装・Stripe接続で遅れ、6/1に間に合わない

**被害**:
- 営業材料不足
- 既存ユーザーの期待感毀損
- LLC設立とのタイミングズレ

**対策**:
1. **スコープ圧縮**: Bilingual(日本語)+Course(手動)+Study Guide(Quick+Std)のみ死守
2. **Deep Mode / Exam Plan / Syllabus自動解析 は 7-8月 Summer 中に追加可**
3. **毎週金曜に進捗確認**、1週間以上ビハインドなら機能削減判断
4. **バックアップ: 6/15 or 6/22 ローンチも OK**（大幅遅延回避）

---

### リスク 2: UF ELI 契約取れない

**シナリオ**: Director 返信なし、デモ後パイロット断られる

**被害**:
- flagship事例失う
- 営業のストーリー構築遅れ

**対策**:
1. **並行で FSU + Santa Fe攻略**
2. **UF内の別ルート**: Japanese Student Association, Academic Coordinator, Accessibility Office
3. **Fallback flagship候補**: Santa Fe College, UCF ISSS, FIU（中南米特化）
4. **"UF前のパイロット校"でも価値** — 「UF系列校で使われてる」でも武器になる

---

### リスク 3: StudyFetch が多言語化

**シナリオ**: Owl Venturesから資金で日本/中国/韓国語版リリース

**被害**:
- 直接競合化、CAC上昇
- バリュエーション圧迫

**対策**:
1. **先行者利益を厚く作る** — UF flagship, 日本人留学生コミュ獲得
2. **機能統合の深さで差別化** — StudyFetchが真似するには時間がかかる
3. **iOSネイティブ優位** — StudyFetchはWebメイン
4. **監視強化**: 月次で StudyFetch公式サイト・App Store更新チェック
5. **買収提案の可能性** — 最悪、Owl/StudyFetchが lecsy を買収

---

### リスク 4: Apple審査リジェクト

**シナリオ**: Stripe誘導パターンがガイドライン違反と判定

**被害**:
- ローンチ遅延 2-4週間
- 既存ユーザーへアップデート配信遅延

**対策**:
1. **メタデータで明記**: "Subscription managed on lecsy.app"
2. **アプリ内に価格表示一切しない**
3. **Reader App 申請検討**（lecsyが該当するか事前確認）
4. **フォールバック**: 万一拒否されたら、Apple Small Business Program (15%) で IAP 導入
5. **Netflix/Spotify 成功事例を援用**: App Review で説明

## 🔄 中優先リスク

### リスク 5: Deepgram 値上げ

**対策**:
- Edge Function で抽象化、AssemblyAI/Soniox 即切替可能に
- Deepgram Growth plan ($4K/年) 到達時に固定レート交渉

### リスク 6: GPT-4o Mini 精度低下・価格変動

**対策**:
- Claude Haiku / Gemini Flash 両対応可能な抽象レイヤ
- Prompt Engineering で精度確保
- Cache 戦略で呼び出し数削減

### リスク 7: 留学生数減少

**シナリオ**: 米国外交政策、ビザ規制強化

**対策**:
- グローバル分散（米一国依存せず）
- Phase 3-5 で UK / EU / 日本市場へ
- 英語非ネイティブネイティブ（米永住者の子弟）も対象化

## 🚨 巨大リスク（低確率だが致命的）

### リスク 8: F-1 コンプラ違反発覚

**シナリオ**: OPT前の無料パイロットが「unauthorized employment」と認定

**被害**:
- ビザ取消
- 強制送還
- 事業停止

**対策**:
1. **ISS への事前相談メール送信・保存** ← **最優先アクション**
2. **Protocol Agreement で "unpaid research" 明記**
3. **金銭授受を一切避ける**（コーヒー・ランチも注意）
4. **移民弁護士に年次レビュー**（$300-500/時）
5. **記録を全保存**（メール・Slackログ）

---

### リスク 9: ソロ運営のバーンアウト

**シナリオ**: 開発 + 営業 + 顧客対応 + 書類作成 を1人で回し続け、3-6ヶ月で疲弊

**被害**:
- プロダクト品質低下
- 営業停滞
- 生活・メンタルヘルス悪化

**対策**:
1. **週1日は絶対休む**（日曜）
2. **運動を習慣化**（週3回以上）
3. **バーチャルアシスタント検討**（$5-10/時で雑務）
4. **メンター月1相談**（水谷明美さん、ヘル、アイゾール）
5. **SDR 早期雇用**（MRR $10K超えたら検討）

---

### リスク 10: キャッシュ枯渇

**シナリオ**: Deepgram残高が尽きる、Stripe手数料で資金繰り悪化

**対策**:
1. **Deepgram 毎日残高チェック**（$500割れたら補充）
2. **Stripe即時振込設定**（週次で現金化）
3. **Year 1 予備費 $5K を別口座**
4. **最悪シナリオ**: lecsyを止めて、Claude Code で副業（$100-200/時可能）
