# Lecsy 現状レポート — 2026-04-15

## 1. 🎯 プロダクトの現在地

**ワンライナー**: 米国留学生向け iOS 講義録音＋文字起こし＋AI要約アプリ。B2B（語学学校・コミカレ）と B2C（個人留学生）の両輪で 2026-06-01 ローンチ。

**現在のコードベース状態**
- **iOS** `lecsy/` — 24 の Services、7 ビュー領域、Swift 6 対応準備中（警告のみ）
- **Web** `web/` — Next.js 14 App Router、42 ルート（LP/ダッシュボード/組織管理/課金）
- **Supabase Backend** — 18 Edge Function、47 migration ファイル
- **Git**: `main` ブランチ clean、最新 `15e14c7`

## 2. 📱 iOS アプリで今できること

### 認証
- **Apple Sign In** / **Google OAuth** / **メール Magic Link（6桁 OTP）**
- Keychain にトークン保存、初回起動で即復元

### 録音（コア機能）
- `RecordView` — 録音ボタン、波形、言語ピッカー、しおり機能、バックグラウンド継続
- **Free**: 録音中は字幕なし → 録音停止後 **WhisperKit** でオンデバイス文字起こし（オフライン可）
- **B2B Pro**: 録音中に **Deepgram Nova-3** で字幕ライブ表示、遅延 <1s
  - `LiveCaptionView` — 単言語ライブ字幕
  - `BilingualCaptionView` — 英語＋母語訳の並列表示（GPT-4o Mini）

### Library（講義一覧）
- `LibraryView` / `LectureDetailView` — 録音再生、トランスクリプト同期ハイライト、スクロール位置復元
- **AI 要約**（`summarize` / `summarize-stream` Edge Function）
  - `mode: 'summary'` — 要点・キーポイント・セクション分割
  - `mode: 'exam_mode'` — 試験対策モード
  - 多言語対応: 日・英・中・韓・西・仏
  - idempotency ガード（10分以内の同一リクエストで再課金防止）

### Settings
- 文字起こし言語選択
- Bilingual 翻訳先言語選択（Pro のみ）
- Plan セクション（Pro via org → 表示 / それ以外 → feature_flag で gate）
- Cloud Sync トグル（トランスクリプトテキストのみアップロード、音声はアップしない）
- Privacy / Terms リンク、アカウント削除、サインアウト

### Organization（B2B）
- `OrganizationTabView` — メンバー一覧、クラス一覧、用語集、利用統計、設定
- **Super Admin** 画面（nittonotakumi@gmail.com 固定）

### Onboarding
- 初回起動時の許可リクエスト（マイク）、言語選択、チュートリアル

## 5. 🔑 プラン判定ロジック

```
PlanService.refresh() が 3 経路を並列フェッチ
  ├─ organization_members.status='active' + org.plan='pro'  → proSource=.organization
  ├─ subscriptions.status='active' + provider='stripe'      → proSource=.stripe
  └─ feature_flags.b2c_stripe_checkout.enabled              → b2cCheckoutEnabled

isPaid = (proSource != .none) && authenticated && !forceLocalTranscription
```

### iOS UI の露出ルール
| ユーザー状態 | 録音中字幕 | 録音後文字起こし | Settings の Plan |
|------------|-----------|----------------|----------------|
| Free (org無し/sub無し) | なし | WhisperKit | 非表示（flag=false時） |
| B2B Pro (org経由) | Deepgram Live | Deepgram Batch | "Pro (via your org)" |
| B2C Pro (stripe経由) | Deepgram Live | Deepgram Batch | "Manage Subscription"（flag=true時） |
