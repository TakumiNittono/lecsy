## 🌍 ワンライナー

> **"Your semester-long AI companion for English lectures."**
>
> 米国大学に通う英語非ネイティブ留学生（80万人市場）が、英語講義を"学期まるごと"攻略する唯一のiOSアプリ。

---

## 🎯 なぜ lecsy が必要か

### 留学生の生の痛点（たくみ自身が経験）
1. 講義で英語が早すぎて聞き取れない → 母語字幕で追従したい
2. 専門用語が初耳すぎて文脈で困惑 → 母語定義が欲しい
3. 1時間の講義を復習する時間がない → 5秒でサマリが欲しい
4. 語彙を覚える時間がない → 自動でAnki化して欲しい
5. 試験対策の計画立てられない → AIにスケジュール組んで欲しい

**1つのアプリで全部解決する製品は世界に存在しない**。

## 💪 5つの構造的アドバンテージ

### 1. たくみ自身が留学生だった
案件解像度が他社より桁違いに高い。ミッションが自然。投資家・顧客に "I built this because I needed it myself" が響く。

### 2. FL在住 × UF徒歩圏
- UF 7,000人国際学生のキャンパスに徒歩アクセス
- UFを flagship にして全米展開の武器にする
- フロリダ = 中南米・カリブ海留学生の重要拠点

### 3. 日本人創業者ストーリー
- アジア留学生市場（日/韓/中/越）で圧倒的信頼
- Plaud / PolyPal / LectMate が取れないブランド面
- SNS・コミュニティ発信が自然（LINE/KakaoTalk/WeChat）

### 4. 大手が構造的に参入できない
- **ChatGPT Edu / Canvas**: 巨大すぎて個別言語に最適化しない
- **Otter / Plaud**: 汎用会議が本業、留学生UXに振れない
- **StudyFetch**: 英語ネイティブ600万獲得済、多言語化でUX毀損
- **Glean**: Disability Office依存、個人B2Cに適応不可

### 5. データ堀が累積
- 大学×科目×学期の専門語彙辞書（Keyterm Prompting）
- 2-3年で真似不可能な資産化
- Deepgram を抽象化して将来乗り換え可能（AssemblyAI/Soniox）

## 🛡️ 競合が lecsy を殺せる唯一の方法

- **StudyFetch が日本/中国語化する**（確率: 5%、Owl Venturesが指示したら）
- **Canvas+OpenAI が留学生向けFeature追加**（確率: 15%、ただし lecsy と競合にならない形）
- **PolyPal が学期OS化+米大学営業**（確率: 20%、Timekettleがハード本業で投資判断遅い）
- **Apple/Google/Microsoft が買収または類似プロダクト発表**（確率: 10%、Plaud買収のほうが先）

→ **最大脅威は StudyFetch**、他は構造的に動けない。

---

## 🎯 絶対に守る5原則

1. **留学生に特化し続ける**。英語ネイティブ向けへの誘惑に負けない
2. **iOS native を死守**。Webは補助、Androidは Phase 3 以降
3. **機能を足すより、1つを磨く**。ローンチ後は売ることに全振り
4. **プライバシー前提を崩さない**。音声はlecsyサーバーに絶対保存しない
5. **日本人創業者ストーリーを隠さない**。最強の差別化武器
