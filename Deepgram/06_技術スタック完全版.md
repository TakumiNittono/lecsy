## 🎯 コア技術選定理由

### Deepgram Nova-3 Multilingual
- **なぜ**: 精度95%+、話者分離、Multilingual autoswitch、低レイテンシ
- **代替**: AssemblyAI（同等精度）、Soniox（多言語強い）
- **決定**: Deepgram採用（コスパ + Python/JS SDK充実 + Keyterm Prompting）
- **料金**: $0.0092/分（Multilingual）= $0.552/時

### GPT-4o Mini
- **なぜ**: 翻訳・要約・Quiz・Vocabulary すべて1モデルで
- **代替**: Claude Haiku 3.5, Gemini Flash
- **決定**: GPT-4o Mini（StreamingとStructured Outputが洗練）
- **料金**: $0.15/M input tokens, $0.60/M output
- **1ユーザー/月**: 翻訳$0.45 + Study Guide$0.25 + Vocab$0.06 + Quiz$0.05 = 約$0.80

### Supabase
- **なぜ**: Postgres + Auth + Storage + Edge Functions + Realtime + pgvector 全部入り
- **代替**: Firebase, AWS Amplify, Neon + Vercel
- **決定**: Supabase（RLS・Edge Functions・pgvector統合）
- **料金**: Pro $25/月、DB/Storage超過で増加

### pgvector
- **なぜ**: 講義全体のセマンティック検索・類似語彙・Professor's Style 分析
- **埋め込み**: text-embedding-3-small（$0.02/M tokens）
- **次元**: 1536（small）
- **インデックス**: ivfflat (cosine similarity)

### Stripe
- **なぜ**: Apple 30%回避、Web直接課金、B2B Invoice対応、多通貨
- **代替**: Paddle（Merchant of Record）, Lemon Squeezy
- **決定**: Stripe（機能・柔軟性・信頼性）

### iOS
- **言語**: Swift 5.10+ / SwiftUI
- **WebSocket**: Starscream 4.0.8+
- **依存**: Supabase Swift SDK, Stripe iOS（使わない方針）
- **OS**: iOS 16+（Live Activities、iOS 17+推奨）

### 3. Vocabulary Intelligence

```typescript
// Edge Function: extract-vocabulary
async function extractVocabulary(
  transcriptId: string,
  userId: string
) {
  const transcript = await fetchTranscript(transcriptId)
  const userLang = await getUserLang(userId)
  
  // GPT-4o Mini: 専門用語抽出
  const terms = await openai.chat.completions.create({
    model: 'gpt-4o-mini',
    messages: [{
      role: 'system',
      content: 'Extract academic vocabulary with definitions...'
    }, {
      role: 'user',
      content: transcript.text
    }],
    response_format: { type: 'json_object' }
  })
  
  // 各用語を embed
  for (const term of terms) {
    const embedding = await openai.embeddings.create({
      model: 'text-embedding-3-small',
      input: term.term + ' ' + term.definition
    })
    
    // 既存辞書との類似度check
    const similar = await supabase.rpc('match_vocabulary', {
      query_embedding: embedding.data[0].embedding,
      match_threshold: 0.8,
      user_id: userId
    })
    
    if (similar.length > 0) {
      // 既出扱い、encounter_count++
    } else {
      // 新規登録
      await supabase.from('personal_vocabulary').insert({
        user_id: userId,
        term: term.term,
        definition_en: term.definition_en,
        definition_native: term.definition_native,
        cefr_level: term.cefr,
        embedding: embedding.data[0].embedding,
        first_encountered_lecture_id: lectureId
      })
    }
  }
}
```

async function generateExamPlan(
  courseId: string,
  examDate: Date,
  userId: string
) {
  const daysUntil = daysBetween(new Date(), examDate)
  const course = await fetchCourseWithLectures(courseId)
  const weaknesses = await detectWeaknesses(userId, courseId)
  const profStyle = await analyzeProfessorStyle(courseId)
  
  // GPT-4o Mini: 学習計画JSON生成
  const plan = await openai.chat.completions.create({
    model: 'gpt-4o-mini',
    messages: [{
      role: 'system',
      content: 'Generate a study plan...'
    }, {
      role: 'user',
      content: JSON.stringify({
        daysUntil, course, weaknesses, profStyle
      })
    }],
    response_format: { type: 'json_object' }

  await supabase.from('study_plans').insert({
    user_id: userId,
    course_id: courseId,
    exam_date: examDate,
    daily_tasks: plan.daily_tasks,
    created_at: new Date()

### 5. 5層安全弁（cost guard）

```typescript
// Edge Function: deepgram-token
async function issueDeepgramToken(userId: string) {
  // L1: Deepgram残高チェック
  const balance = await checkDeepgramBalance()
  if (balance < 100) return reject('Service unavailable')
  
  // L2: 月次cap
  const monthlyUsage = await getMonthlyUsage(userId)
  const planLimit = await getPlanMonthlyLimit(userId)
  if (monthlyUsage >= planLimit) return reject('Monthly limit reached')
  
  // L3: 日次cap
  const dailyUsage = await getDailyUsage(userId)
  const dailyLimit = await getPlanDailyLimit(userId)
  if (dailyUsage >= dailyLimit) return reject('Daily limit reached')
  
  // L4/L5 は iOS side
  
  return generateShortLivedToken({ ttl: 15 * 60 })
}
```

## 💾 データモデル要点

### transcripts（中核）
```sql
create table public.transcripts (
  id uuid primary key default gen_random_uuid(),
  user_id uuid references auth.users(id),
  lecture_id uuid references public.lectures(id),
  language text not null,
  duration_sec int not null,
  words jsonb not null,  -- Deepgram word-level
  translated_sentences jsonb,  -- 母語訳
  embedding vector(1536),  -- セマンティック検索用
  recorded_at timestamptz not null,
  created_at timestamptz default now(),
  text_search tsvector generated always as (
    to_tsvector('simple', 
      (select string_agg(w->>'word', ' ') 
       from jsonb_array_elements(words) w))
  ) stored
);
```

### Course Hierarchy（4階層）
```sql
semesters → courses → weeks → lectures
```

### Vocabulary（pgvector使用）
```sql
create table public.personal_vocabulary (
  id uuid primary key default gen_random_uuid(),
  user_id uuid references auth.users(id),
  term text not null,
  definition_en text,
  definition_native text,
  cefr_level text,
  embedding vector(1536),
  encounter_count int default 1,
  next_review_at timestamptz,
  created_at timestamptz default now()
);
create index on public.personal_vocabulary 
  using ivfflat (embedding vector_cosine_ops);
```

## 🚨 技術的落とし穴

1. **Deepgram WebSocket の再接続**: ネットワーク不安定でconnection drops、自動retry必須
2. **GPT-4o Mini の JSON 出力**: `response_format: { type: 'json_object' }` 必ず指定
3. **pgvector のインデックス**: 100k行超えるとivfflatよりhnsw検討
4. **iOS バックグラウンド録音**: AVAudioSession の適切なカテゴリ設定必要
5. **既存WhisperKitデータの移行**: iOSアプリ更新時、データマイグレーション冪等性
6. **多通貨Stripe Price**: 各通貨ごとにPrice object作る（自動変換ではない）
