## 🎯 戦略サマリ

**B2C留学生個人で実績→同大学のISSS B2Bへアップセル→Enterprise全学契約** の3段階。

```
Phase 1 (2026 Q2-Q3): 
  B2C個人留学生獲得
  → 1,000-3,000人 (日本/韓国/中国/ベトナム系)

Phase 2 (2026 Q4-2027 Q1):
  ISSS B2B販売（$5K-25K/年/校）
  → "学生の N% が既に使ってる" で営業
  → UF/USF/UCF/FIU 4校契約目標

Phase 3 (2027 Q2-Q4):
  Enterprise全学契約（$25-75K/年）
  → ISSS → Disability Office → 全学Accessibility予算
  → 米国Top 20国際学生数大学を狙う
```

## Phase別ターゲット詳細

### Phase 1: B2C個人留学生（2026 Q2-Q3）

**狙い**: 実績ベースを作り、Phase 2のB2B交渉材料にする

#### 優先セグメント1: 日本人留学生（最初の1000人）
- **なぜ最初か**: たくみが日本語ネイティブ、SNS発信が自然
- **獲得チャネル**:
  - 日本人留学生向けTwitter/X/Instagram
  - LINE留学コミュニティ
  - 日本の留学エージェント（5月から交渉）
  - JASSOブログ・記事掲載
  - YouTuber留学vlog 2-3人とコラボ
- **目標**: 2026年末までに日本人ユーザー 500-1,000人
- **ARPU**: Pro $12.99 or Student $6.99

#### 優先セグメント2: 韓国人留学生（次の500人）
- **なぜか**: 在米42,000人、EdTechへの支出高い
- **獲得**: KakaoTalk留学生コミュ、韓国人YouTubeコラボ
- **目標**: 2027年Q1までに韓国人500人

#### 優先セグメント3: 中国人留学生（大規模だが難しい）
- **なぜか**: 在米259,000人、最大市場
- **難点**: LectMate（Youdao）が既に定番、中国本土でアクセス問題
- **アプローチ**: WeChat留学生グループ、小红书、米国内中国人コミュのみ
- **目標**: 2027年末までに中国人1,000人

#### セグメント4以降: ベトナム・ブラジル・メキシコ
- **2027年Q2以降**、言語UI追加と並行

#### ISSS Starter $499/月プランの営業論理
- 50-100席 → $5-10/席/月 = 学生あたり年$60-120
- これはDisability支援よりは安いがISSSでは現実的
- "既に N人の学生が個人契約してる" が決定打

#### アプローチ手順（詳細）

**Step 1: Discovery（Week 1-2）**
- ISSS Director特定（LinkedIn + 公式サイト）
- メール or LinkedIn DMで「既に貴校のN人の留学生が使ってます」打診
- 15分Zoom

**Step 2: パイロット（Week 3-12）**
- 無料パイロット 6-12週間、20-50学生
- 効果測定（利用時間、満足度、学業成績相関）

**Step 3: Contract（Week 12-16）**
- 正式契約、Stripe Invoice、Fall semester対応
- LTI/SAMLはEnterprise化時に追加

## 🎯 なぜ ISSS が Disability Office より先か

| 項目 | ISSS | Disability Office |
|-----|------|----------------|
| 予算規模 | $50-200K/校 | $200K-1M/校 |
| 意思決定速度 | 2-4ヶ月 | 6-12ヶ月 |
| 調達プロセス | 軽い | HECVAT/DPA/VPAT必須 |
| 必要書類 | 軽量 | 重い |
| **lecsy初期契約しやすさ** | **★★★★★** | ★★★ |

→ **ISSSで実績作ってからDisability巻き込む**が正解。
