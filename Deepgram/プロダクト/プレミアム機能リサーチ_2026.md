## 1. AI語彙抽出 & フラッシュカード自動生成

### 概要
講義トランスクリプトからキーワード・専門用語を自動抽出し、定義付きのAnki形式フラッシュカードを生成。

### API/サービス比較

| サービス | 方式 | コスト | 品質 | 備考 |
|---------|------|--------|------|------|
| **OpenAI GPT-4o-mini** | プロンプトで抽出+定義生成 | ~$0.15/1Mトークン（input） | ◎ | **推奨。既にAPI契約あり** |
| **Claude 3.5 Haiku** | 同上 | ~$0.25/1Mトークン（input） | ◎ | GPT-4o-miniの代替 |
| **spaCy + カスタムNER** | オンデバイスNLP | 無料（OSS） | △ | 精度が低い。教育ドメイン辞書が必要 |
| **Google NotebookLM方式** | Gemini 2.5 Flash | $0.15/1Mトークン | ◎ | 2025年9月にフラッシュカード機能追加 |

### 推奨実装
```
トランスクリプト → GPT-4o-mini（用語抽出プロンプト）
  → JSON [{term, definition, example_sentence, difficulty_level}]
    → iOS側でAnki風UI表示
    → Supabase glossary_entries テーブルに保存（既存スキーマ流用）
```

### 実装詳細
- **難易度**: Easy（2-3日）
- **コスト**: 1講義（5,000語）あたり約$0.002
- **月間100講義**: 約$0.20
- **課金ポイント**: Pro $9.99/月の目玉機能
- **競争優位**: Otter/Nottaにはない。ESL特化の語彙レベル分類（CEFR A1-C2）が差別化

### 収益ポテンシャル
- B2C: Pro課金（$9.99/月）の主要機能
- B2B: 語学学校が最も欲しい機能の1つ。「テスト対策の単語帳が自動で作れる」は刺さる
- **月間追加収益見込み: B2Cで$500-$1,500、B2Bで$1,000-$3,000**

## 2. AI講義要約 & スタディノート

### 概要
トランスクリプトから構造化されたスタディノート、要点リスト、クイズ問題を自動生成。

### API/サービス比較

| サービス | コスト | 品質 | 特徴 |
|---------|--------|------|------|
| **OpenAI GPT-4o-mini** | $0.15/1M input, $0.60/1M output | ◎ | **推奨。既にEdge Functionで実装済み** |
| **OpenAI GPT-4o** | $2.50/1M input, $10/1M output | ◎◎ | 長い講義（>30分）で精度向上 |
| **Claude 3.5 Sonnet** | $3/1M input, $15/1M output | ◎◎ | 長文コンテキストに強い（200K） |
| **Claude 3.5 Haiku** | $0.25/1M input, $1.25/1M output | ◎ | コスパ最強の代替 |
| **DeepSeek API** | $0.07/1M input | ○ | 安いが品質ムラあり |

### 推奨実装
```
トランスクリプト → GPT-4o-mini（要約プロンプト）
  → 3階層出力:
    1. 1行サマリ
    2. 3-5行の要点リスト
    3. 詳細ノート（見出し+箇条書き）
    4. クイズ問題（5問、多肢選択 + 記述）
```

### 実装詳細
- **難易度**: Easy（Edge Function既存。iOS接続で1-2日）
- **コスト**: 1講義（50分 = ~8,000語）あたり約$0.01-0.03
- **課金ポイント**: Pro課金の核心機能
- **競争優位**: Mindgrasp（$11.99/月）、NoteGPT（$9.99/月）が専用アプリ。lecsyは「録音→文字起こし→要約」がシームレス

### クイズ生成の差別化
- ESL向け: 聞き取り問題（音声再生+穴埋め）
- CEFR レベル別出題
- Bloom's Taxonomy対応（記憶/理解/応用/分析）
- **これは投資家が「adaptive learning」と呼ぶもの**

### 収益ポテンシャル
- **月間追加収益見込み: B2Cで$500-$2,000、B2Bで$2,000-$5,000**

## 3. 発音アセスメント / スピーキング練習

### 概要
学生の発音をネイティブスピーカーと比較し、音素レベルでスコアリング。

### API/サービス比較

| サービス | 価格 | 品質 | 言語数 | 特徴 |
|---------|------|------|--------|------|
| **Azure Speech 発音評価** | **$1.02-1.32/時間** | ◎◎ | 28言語 | **最推奨。音素/単語/流暢さ/韻律を評価。教育向けチュートリアルあり** |
| **SpeechAce** | カスタム見積り（$50-500/月〜） | ◎◎ | 英語特化 | 発音+流暢さ+文法を統合評価。教育特化 |
| **ELSA API** | カスタム見積り | ◎◎ | 英語のみ | +95%精度。IELTS/TOEFL予測スコア。非ネイティブ音声に最適化 |
| **Google Speech-to-Text** | $0.006/15秒 | ○ | 125+言語 | 発音評価機能なし。認識のみ |

### Azure Speech 発音評価の詳細
```
入力: 音声 + リファレンステキスト（講義の一部）
出力:
  - AccuracyScore（正確さ）: 0-100
  - FluencyScore（流暢さ）: 0-100
  - CompletenessScore（完全性）: 0-100
  - ProsodyScore（韻律）: 0-100
  - 音素レベルのフィードバック
```

### 推奨実装
- Phase 1: Azure Speech（多言語対応、lecsyの9言語と親和性高い）
- Phase 2: ELSA API連携（英語特化の高精度フィードバック）

### 実装詳細
- **難易度**: Medium（2-3週間。音声録音UIの拡張 + APIコール + 結果表示）
- **コスト**: 学生1人が1日5分練習 × 30日 = 150分/月 → 約$2.50/学生/月
- **B2B 50人クラス**: 約$125/月のAPI費用
- **課金ポイント**: B2B Growth以上のプランに含める。B2C Pro機能

### 競争優位
- **ELSA**: 発音練習「だけ」のアプリ（$11.99/月）。文字起こしなし
- **lecsy**: 講義録音→文字起こし→発音練習が一気通貫。**講義で出た単語の発音を練習できる**のは唯一
- **これがlecsyの殺し文句**: 「今日の講義のキーワードを発音練習しましょう」

### 収益ポテンシャル
- B2B: 語学学校が最も価値を認める機能。契約単価$500-$2,000/月上乗せ可能
- B2C: Pro+プラン（$14.99/月）の差別化要素
- **月間追加収益見込み: $1,000-$5,000**

## 5. AI チューター / 講義Q&A（RAG）

### 概要
学生が講義内容について質問すると、トランスクリプトを検索してAIが回答。

### 技術スタック

| コンポーネント | 推奨 | コスト | 備考 |
|-------------|------|--------|------|
| **Embedding** | OpenAI text-embedding-3-small | $0.02/1Mトークン | 1536次元。コスパ最強 |
| **Vector DB** | **Supabase pgvector** | $0（既存Pro契約に含む） | **既にSupabase使用中。追加コストゼロ** |
| **LLM** | GPT-4o-mini | $0.15/1M input | 回答生成 |
| **Orchestration** | LangChain.js or 自前 | 無料 | Next.js Edge Functionで動く |

### 推奨アーキテクチャ
```
1. トランスクリプトを500トークンずつチャンク分割
2. text-embedding-3-small でベクトル化
3. Supabase pgvector に保存
4. 学生の質問をベクトル化 → コサイン類似度検索
5. 上位5チャンクをコンテキストとして GPT-4o-mini に渡す
6. 回答 + ソース（何分何秒の発言か）を返す
```

### コスト試算
- 1講義のインデキシング: ~$0.001（embedding）
- 1質問あたり: ~$0.002（embedding + completion）
- 学生50人 × 1日5質問 × 30日 = 7,500質問/月 → **約$15/月**

### 実装詳細
- **難易度**: Medium（2-3週間。Supabase pgvector拡張 + Edge Function + iOS UI）
- **課金ポイント**: B2B Growth以上。B2C Pro。**「24時間AIチューター」は最強の売り文句**
- **競争優位**: 極めて高い。「自分の講義について質問できる」アプリは存在しない

### 投資家ウケ
- **2026年のEdTech投資で最もホットなキーワード = 「AI Tutor」「Agentic AI」**
- Gartner: 2026年末までにエンタープライズアプリの40%がAIエージェント搭載
- **lecsy + RAG = 「講義に特化したAIチューター」。これはシード資金調達の核になる**

### 収益ポテンシャル
- B2B: Enterprise契約の決め手。$500-$2,000/月の追加価値
- B2C: Pro+プラン（$14.99/月）への引き上げ
- **月間追加収益見込み: $2,000-$8,000**

| サービス | 価格 | 機能 | 備考 |
|---------|------|------|------|
| **Azure Text Analytics** | $1/1,000テキストレコード | 感情分析、キーフレーズ抽出 | 多言語対応 |
| **OpenAI GPT-4o-mini** | $0.15/1M input | 感情+理解度推定 | プロンプトで柔軟にカスタム |
| **AWS Comprehend** | $0.0001/ユニット | 感情分析、エンティティ認識 | バッチ処理向き |
| **自前集計** | $0 | 録音時間、頻度、ブックマーク数 | **まずこれだけで十分** |

### 推奨実装（段階的）
```
Phase 1（今すぐ）: 既存データで基本分析
  - 学生別: 録音回数、総時間、言語分布、ブックマーク数
  - クラス別: アクティブ率、平均録音時間
  - → B2B Webダッシュボードに表示

Phase 2（Year 2）: AI分析追加
  - トランスクリプトの感情分析（Azure/GPT-4o-mini）
  - 「理解度スコア」（質問頻度 + 復習頻度 + クイズ正答率）
  - 教師へのウィークリーレポート自動生成
```

### 実装詳細
- **Phase 1 難易度**: Easy（1週間。SQLクエリ + Webダッシュボード）
- **Phase 2 難易度**: Medium（2-3週間）
- **コスト**: Phase 1は$0。Phase 2はGPT-4o-miniなら月$10-30
- **課金ポイント**: B2B Growth/Enterpriseの差別化。教師が「うちの生徒の使い方が見える」
- **競争優位**: Otter/Nottaにはない。SmartClassのラボ機能に近いが、lecsyはソフトウェアのみ

### 投資家ウケ
- **「measurable learning outcomes」は2026年のVC必須チェック項目**
- エンゲージメント分析 = 「lecsyは学習効果を数値で証明できる」
- **NRR（Net Revenue Retention）を高める要素**: 教師がデータに依存すると解約しない

### 収益ポテンシャル
- B2B契約単価を$100-$300/月引き上げ
- **月間追加収益見込み: $500-$2,000**

## 7. LMS連携（Canvas, Blackboard, Google Classroom）

| LMS | 連携方式 | 難易度 | 市場シェア（米大学） |

```
必要なもの:
  - OAuth 2.0 + JWT認証
  - JWKS エンドポイント
  - Deep Linking（課題にlecsyコンテンツを埋め込む）
```

### 実装詳細
- **難易度**: Hard（Canvas LTI 1.3で3-4週間）
- **コスト**: 開発コストのみ（LTI自体は無料規格）
- **課金ポイント**: B2B Enterprise の必須条件。大学は「LMS連携なし = 導入不可」
- **競争優位**: Otter/NottaはLTI非対応。Panopto/Echo360は対応済み

### 収益ポテンシャル

- **年間追加収益見込み: $50K-$200K（大学1校の契約額）**

## 8. アクセシビリティ機能

### 概要
ADA Title II（2025年4月施行）により、教育機関はデジタルアクセシビリティ準拠が法的義務化。

### 機能別の実装

| 機能 | 方式 | コスト | 難易度 |
|------|------|--------|--------|
| **ディスレクシア対応フォント** | OpenDyslexic / Lexend フォントを設定に追加 | 無料（OSSフォント） | **Easy（半日）** |
| **Apple Accessibility Reader連携** | iOS 19のシステム機能を活用 | 無料 | Easy |
| **VoiceOver最適化** | SwiftUI Accessibility modifier | 無料 | Easy-Medium（1週間） |
| **フォントサイズ・色・行間カスタマイズ** | iOS設定UI | 無料 | Easy（2-3日） |
| **手話アバターオーバーレイ** | SignAvatar等（研究段階） | 不明 | **Hard（技術未成熟）** |
| **ハイコントラストモード** | iOS標準 + カスタムテーマ | 無料 | Easy |

### 推奨実装（コスパ順）
1. **ディスレクシアフォント追加**（半日。差別化効果大）
2. **VoiceOver完全対応**（1週間。ADA準拠の法的要件）
3. **フォント/色/行間カスタマイズ**（2-3日）
4. **ハイコントラストモード**（1日）

### 手話アバターについて
- 2025年時点で商用レベルのAPIは存在しない
- SignAvatar（signavatar.org）が研究レベルで存在
- **実用化は2-3年先。今は手を出さない**
- 代替: 「文字起こし自体がろう者のアクセシビリティ」。lecsyの核機能がすでにアクセシビリティ

### 営業での使い方
- **ADA対応は大学契約のチェックリスト項目**
- 「lecsy はADA Title II準拠（文字起こし + ディスレクシアフォント + VoiceOver対応）」と言える状態を作る
- HECVAT（大学セキュリティ評価）にもアクセシビリティ項目がある

### 収益ポテンシャル
- 直接的な課金ポイントではない。**大学契約の「門番」をパスするための必須要件**
- ADA非準拠 = 大学は導入できない = 年間$15K-$200Kの契約を失う
- **投資対効果: 2-3日の作業で大学市場への門を開く**

## 9. 音声合成（TTS）/ 講義リプレイ

### 概要
トランスクリプトをTTSで読み上げ。速度変更、別の声で聞き直し。

### API/サービス比較

| サービス | 価格/1Mキャラ | 品質 | 無料枠 | 特徴 |
|---------|-------------|------|--------|------|
| **OpenAI TTS (tts-1)** | **$15** | ◎ | なし | **推奨。品質/コストのバランス最良** |
| **OpenAI TTS (tts-1-hd)** | $30 | ◎◎ | なし | 高品質版 |
| **Google WaveNet** | $4 | ○ | **4Mキャラ/月** | 無料枠が巨大。開発中に最適 |
| **Amazon Polly Neural** | $16 | ○ | 5Mキャラ/12ヶ月 | AWS既存ユーザー向け |
| **Azure Neural TTS** | $16 | ◎ | 500Kキャラ/月 | 多言語に強い |
| **ElevenLabs** | $165-180 | ◎◎◎ | 10Kキャラ/月 | 最高品質だが**10倍高い** |

### コスト試算
- 1講義（50分）のトランスクリプト ≈ 40,000キャラ
- OpenAI tts-1: 約$0.60/講義
- Google WaveNet: 約$0.16/講義（無料枠で月100講義まで無料）

### 推奨実装
```
Phase 1: Apple AVSpeechSynthesizer（オンデバイス、無料、即実装）
  → 品質は低いが、オフライン対応 & コストゼロ

Phase 2: OpenAI TTS or Google WaveNet
  → Pro課金ユーザーのみ。高品質な読み上げ
  → 速度調整（0.5x〜2.0x）
  → 9言語対応（lecsyの言語キットと連動）
```

### 実装詳細
- **Phase 1 難易度**: Easy（1-2日。AVSpeechSynthesizer）
- **Phase 2 難易度**: Medium（1週間。API連携 + 音声キャッシュ）
- **課金ポイント**: Pro機能。「講義を聞き直す」はESL学生の核心ニーズ
- **競争優位**: Otter/Nottaは元音声再生のみ。TTSで「別の速度/声で聞き直し」は差別化

### 収益ポテンシャル
- Pro課金のバンドル機能として
- **月間追加収益見込み: 直接$0だが、Pro課金の解約防止効果**

### 概要

## EdTech投資家が2025-2026年に求めるもの

### 市場概況
- 2025年のEdTech VC資金: 約$2.6B（前年比やや減少）
- 2026年のEdTech市場規模: 約$12.6B
- **トレンド**: ディール数は減少、1件あたりの投資額は増加（平均$7.8M）

### 投資家が最も重視する5要素

| # | 要素 | 詳細 | lecsyへの示唆 |
|---|------|------|-------------|
| 1 | **Efficacy Proof（効果の証明）** | 学習成果が10%以上向上するエビデンス。Logic ModelまたはA/Bテスト結果。**証明なし = 30-50%のバリュエーション割引** | Santa Feでのパイロットデータを収集。「lecsy使用クラス vs 非使用クラス」の成績比較 |
| 2 | **Agentic AI** | 単なる生成AIラッパーではなく、自律的にワークフローを実行するAI。スキルギャップ検出→介入→成績更新を自動化 | RAG + 語彙抽出 + クイズ自動生成を統合して「AIチューターエージェント」と位置付ける |
| 3 | **NRR（Net Revenue Retention）** | 既存顧客の拡大率。120%+が理想 | エンゲージメント分析で教師の依存度を高め、解約を防ぐ |
| 4 | **LTI 1.3統合** | 大学調達の必須条件。LTI非対応 = 調達候補にすら入らない | Phase Cで実装予定。投資家ピッチでは「LTI 1.3対応済み」と言えることが重要 |
| 5 | **コンプライアンス自動化** | FERPA/COPPA 2.0/ADA Title IIへの準拠。「コンプライアンスが機能として組み込まれている」 | オンデバイス処理 = FERPA的に最強。これをもっと前面に出す |
