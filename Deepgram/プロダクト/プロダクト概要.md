## 🎯 ビジョン

> **"Your semester-long AI companion for English lectures."**
>
> 米国大学に通う英語非ネイティブ留学生（80万人市場）が、英語講義を"学期まるごと"攻略するための世界で唯一のiOSアプリ。

---

## 🔑 ポジショニング（新・確定）

**競合は存在するが、"全部を統合した留学生向けiOS native"は世界に存在しない:**

| 競合 | 強み | lecsyとの違い |
|-----|-----|----------|
| PolyPal (Timekettle) | Bilingual字幕 | 翻訳のみ、学習/コース管理なし |
| LectMate (Youdao) | 中国人留学生定番 | 中国語UIのみ、アジア系以外不在 |
| StudyFetch ($11.5M調達) | AI Study Guide | **英語ネイティブ専用、翻訳未対応** |
| Glean (旧Sonocent) | 大学Disability Office標準 | UI古い、学習AI薄い、留学生向けでない |
| Otter.ai | ブランド | 汎用会議、留学生UX無し |
| Canvas + OpenAI | LMS統合 | 巨大すぎて留学生特化できない |

**lecsyは唯一**、これら全部の"強みだけを取って統合した"留学生専用iOSアプリ。

## 🚀 5つのキラー機能（全部"最高峰"）

詳細仕様: [[キラー5機能仕様]]

### ① 🌐 Real-time Bilingual Captions
- Deepgram Nova-3 Multilingual、遅延 <500ms
- 英語原文 + 母語訳の側々表示
- 話者分離、タップ辞書、スクロールバック、専門用語ハイライト
- **PolyPal/LectMateを精度・レイテンシ・UXで全て上回る**

### ② 📚 Course Hierarchy
- 学期 → コース → 週 → 回の4層階層
- シラバスPDF → GPT-4o Miniが自動パース
- Canvas/Moodle/Blackboard LTI連携（Phase 2）
- **Glean的だが現代UI、競合空白の中心領域**

### ③ 🤖 AI Study Guide + Quiz
- 講義終了5秒で生成（Quick / Standard / Deep の3階層）
- Bilingual Q&A（母語質問 → 英語+母語回答+講義タイムスタンプ）
- **StudyFetchを留学生向けで上回る唯一の製品**

### ④ 📖 Vocabulary Intelligence → Anki連携
- リアルタイム専門用語抽出、Context保存
- CEFR判定、Spaced Repetition内蔵
- Anki/Quizlet/Notion one-tap export
- **誰もやってない**

### ⑤ 🎯 Exam Prep Plan
- 試験日入力 → 自動で2週間の学習計画生成
- シラバス連動、Professor's Style 検出、Daily Push通知
- **完全な競合空白、lecsy独占領域**

├─ /for-universities — ISSS向け
└─ /dashboard — B2B管理

## 🛡️ 競合がlecsyを真似できない構造的理由

1. **ChatGPT Edu / Canvas**: 巨大すぎて個別言語チューニングに振れない
2. **Otter / Plaud**: 汎用ビジネス会議が本業、留学生UXに全振りできない
3. **StudyFetch**: 英語ネイティブ600万獲得済、今さら多言語化できない
4. **Glean**: Disability Office営業構造、個人留学生B2Cに適応不可
5. **PolyPal / LectMate**: 翻訳ハード副産物 / 中国市場依存、学期OS化できない

**→ lecsyの堀:**
- 留学生言語別コミュ（LINE/WeChat/KakaoTalk）のクチコミ優位
- 大学×科目×学期の専門語彙辞書の累積データ資産
- 日本人起業家ストーリーによる共感信頼
- FL ESL学校営業資産 → ISSSへの橋
