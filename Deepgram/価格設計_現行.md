## 🎯 設計原則

1. **Deepgramコストと翻訳コストを分離して課金gateを張る**
   - Deepgram (STT): 全プラン開放（Freeも英語ライブ字幕を体験可能）
   - GPT-4o Mini (翻訳): Pro/Student/Powerのみ（Bilingualの本質的差別化）
   - これで Free ユーザーもコアバリュー「講義中に助かる」を体験 → 翻訳欲しさで課金へ

2. **Apple IAP 不採用 / Stripe Web Checkout 一本**
   - 30%手数料回避、Student $7.99を黒字化
   - B2B Invoiceも同じStripeで処理可能

3. **4層構造（Decoy Effect）**
   - Free / Student / Pro / Power の4段
   - Power がUpper-Anchorとして機能 → Pro が"お得な真ん中"に見える心理設計

4. **キャンペーン期間中は全員 Pro扱い**
   - ～2026-06-01 00:00 JST
   - Stripe課金導線は動くが、UI上は「🎉 Free for everyone」バナー
   - 6/1到達で自動的に本価格設計へ切り替わる（コード変更不要）

## 🔧 実装ロジック（deepgram-token Edge Function）

```typescript
// プラン判定
let plan: 'free' | 'student' | 'pro' | 'power' = 'free';

if (Date.now() < CAMPAIGN_END_MS) {
  plan = 'pro';  // キャンペーン中は全員Pro
} else {
  // 1. Stripe subscription 確認
  const sub = await supabase.from('subscriptions').select('status, price_lookup_key')
    .eq('user_id', userId);
  if (sub?.status === 'active' || sub?.status === 'trialing') {
    if (sub.price_lookup_key.startsWith('lecsy_power')) plan = 'power';
    else if (sub.price_lookup_key.startsWith('lecsy_student')) plan = 'student';
    else plan = 'pro';
  }

  // 2. B2B org member 確認
  if (plan === 'free') {
    const orgMember = await supabase.from('organization_members')
      .select('organizations!inner(plan)')
      .eq('user_id', userId).eq('status', 'active');
    if (orgMember?.organizations?.plan === 'pro') plan = 'pro';
  }
}

// Cap定義
const PLAN_CAPS = {
  free:    { dailyMinutes: 30,  monthlyMinutes: 120 },  // 2h/月
  student: { dailyMinutes: 120, monthlyMinutes: 600 },  // 10h/月
  pro:     { dailyMinutes: 180, monthlyMinutes: 900 },  // 15h/月
  power:   { dailyMinutes: 240, monthlyMinutes: 1800 }, // 30h/月
};
```

### 翻訳gate（translate-realtime Edge Function）

```typescript
if (plan === 'free') {
  return json({
    error: 'plan_upgrade_required',
    feature: 'translation',
    suggestedPlan: userEmail.endsWith('.edu') ? 'student' : 'pro',
  }, 403);
}
```

### iOS側の反応

`TranslationStream` が `plan_upgrade_required` を受信:
1. `upgradeRequired = true` に設定
2. `BilingualCaptionView` の右カラムが CTA "Unlock live translation" → "View Plans" ボタン
3. `suggestedPlan` に応じて Student or Pro を強調表示
4. `BillingService.shared.openPricing()` で Stripe Checkout へ誘導

左カラム（英語ライブ字幕）は Free でもそのまま表示し続ける。

## 💵 粗利構造（Rev.2 修正後）

### B2C（Stripe 2.9% + $0.30 / 平均利用時間想定）

| プラン | 売上 | Stripe | Deepgram | GPT | Supabase | 粗利 | 率 |
|-------|-----|--------|----------|-----|----------|------|---|
| Free ×1人 | $0 | — | **-$1.10**（2h×$0.55） | — | -$0.05 | **-$1.15** | 獲得投資 |
| Student $7.99 | $7.99 | -$0.53 | -$3.96（6h×$0.66） | -$0.40 | -$0.30 | **$2.80** | **35%** |
| Pro $12.99 | $12.99 | -$0.68 | -$5.28（8h×$0.66） | -$0.81 | -$0.30 | **$5.92** | **46%** |
| Power $24.99 | $24.99 | -$1.02 | -$9.90（15h×$0.66） | -$1.50 | -$0.30 | **$12.27** | **49%** |

### Free コスト暴走阻止の計算
- Free ユーザー 1人 = 月 120min 上限 × Deepgram $0.0092/min = **$1.10/月が最大コスト**
- Free ユーザー 1000人全員が上限張り付き = **$1,100/月** の獲得投資
- その 3% (30人) が Student に転換 = $7.99 × 30 = **$240/月** → 最大コストの 22%回収
- その 2% (20人) が Pro に転換 = $12.99 × 20 = **$260/月** → 合計 45%回収
- **残り $600/月は純粋な獲得投資**。LTV で回収する前提

### 旧版との比較（Rev.1 → Rev.2）
| 指標 | Rev.1 (Free 5h) | Rev.2 (Free 2h) | 改善 |
|----|----|----|----|
| Free 1人あたり出血 | $2.80/月 | $1.15/月 | **59%削減** |
| Free 1000人の出血 | $2,800/月 | $1,100/月 | **60%削減** |
| Student粗利率 | 26% | **35%** | 9pt改善 |
| Power追加ARPU影響 | - | +10%（ポートフォリオMix） | - |

### コスト暴走阻止
1. `deepgram-balance-check` (6時間ごと cron): 残高 < $100 で `realtime_captions_beta` flag を OFF
2. 月次cap 120min/ユーザー(Free) / 600min(Student) / 900min(Pro) / 1800min(Power) のハードリミット
3. 日次cap 30min(Free) / 120min(Student) / 180min(Pro) / 240min(Power) で1日で使い切るのを防ぐ
4. Free→Student/Pro 転換率モニタリング: 月次KPIダッシュボードで <2% なら Free時間をさらに絞る

## 📈 損益分岐ライン（Rev.2）

### B2C単独のbreak-even
- 固定費想定: Supabase $50 + Deepgram balance reserve $200 + ドメイン等 $50 = **$300/月**
- 必要粗利: $300
- 必要契約数例:
  - Student 108人（$2.80×108=$302）
  - Pro 51人（$5.92×51=$302）
  - Power 25人（$12.27×25=$307）
  - 現実はMix: Student 50 + Pro 20 + Power 5 = $280 → ほぼ到達

### B2B単独のbreak-even
- **ISSS Starter 1契約 ($599/月) 取れた瞬間、固定費回収 + 黒字化**
- Growth 1契約 = Starter 2.5本分
- Premium 1契約 = Pro個人契約 **675件相当**

→ **B2B 1契約 = B2C 100人分の売上**。営業優先順位の根拠。
