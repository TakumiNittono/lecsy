# lecsy - プロジェクト概要

## 🎯 プロダクトビジョン

**lecsy**は、大学の講義音声を**完全無料・完全ローカル**で録音・文字起こしし、Webで高度なAI機能を提供する統合学習プラットフォームです。

---

## ✨ コアバリュー

| 価値 | 説明 |
|------|------|
| **完全無料録音** | 音声録音・文字起こしは完全無料・無制限 |
| **完全ローカル** | 音声はデバイス内で処理、サーバー送信なし |
| **オフライン動作** | ネットワーク不要で録音・文字起こし可能 |
| **Web連携** | ユーザー操作でテキストのみをクラウド保存 |
| **AI学習支援** | Pro課金でAI要約・Exam Modeを提供 |

---

## 🔐 プライバシー原則

1. **音声はサーバーに送らない** - ローカル処理のみ
2. **自動同期しない** - ユーザー操作でのみ送信
3. **テキストのみ送信** - 音声ファイルは端末内に留まる
4. **完全なデータ隔離** - RLSによりユーザー間のデータ分離
