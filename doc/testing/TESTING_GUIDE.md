# テストガイド

## Part 1: MVP 機能テスト チェックリスト

### 録音機能
- [ ] 初回起動時にマイク権限リクエストが表示される
- [ ] 録音ボタンで録音開始・停止ができる
- [ ] 録音中はストップウォッチが動作する
- [ ] Live Activity（ロック画面・Dynamic Island）に録音状態が表示される
- [ ] バックグラウンド・ロック画面でも録音が継続する

### 文字起こし機能
- [ ] 初回文字起こし時にモデルが自動ダウンロードされる（進捗表示あり）
- [ ] 録音停止後に自動的に文字起こしが開始・完了する
- [ ] 設定画面で言語変更ができる（自動/日本語/英語）
- [ ] 失敗時にエラー状態が表示され、再試行できる

### データ管理
- [ ] LibraryViewに講義一覧が表示される（タイトル・日時・状態）
- [ ] 検索バーでタイトル・テキスト検索ができる
- [ ] 講義詳細でタイトル編集・テキストコピーができる
- [ ] スワイプで講義を削除できる

### 認証機能
- [ ] Google / Apple ログインが正常に完了する
- [ ] ログアウトが正常に動作する
- [ ] アプリ再起動後もログイン状態が維持される

### Web同期機能
- [ ] ログイン済みで「Save to Web」が成功する
- [ ] 未ログイン・ネットワークエラー時に適切なメッセージが表示される
- [ ] 「Retry Uploads」で保留中のアップロードを再試行できる

## Part 3: Stripe 課金フローテスト

### 前提条件
- Stripe Test mode が有効
- 環境変数が設定済み（`ENV_SETUP.md` 参照）

### テスト用カード番号

| カード番号 | 結果 |
|-----------|------|
| `4242 4242 4242 4242` | 成功 |
| `4000 0000 0000 3220` | 3Dセキュア認証が必要 |
| `4000 0000 0000 0002` | カード拒否 |

有効期限: `12/34`, CVC: `123`, 郵便番号: `12345`

### テストフロー

1. Webアプリにログイン > ダッシュボード(`/app`)
2. Subscription カードが「Free」であることを確認
3. 「Upgrade to Pro — $2.99/mo」をクリック
4. Stripe Checkout でテストカード入力 > Subscribe
5. `/app?success=true` にリダイレクト
6. トースト通知表示、Subscription カードが「Pro」に変化
7. `subscriptions` テーブルに `status: "active"` のレコードを確認

### Webhook 動作確認

```bash
stripe listen --forward-to http://localhost:54321/functions/v1/stripe-webhook
```

ターミナルに `checkout.session.completed` イベントと `[200]` レスポンスが表示されることを確認。

### サブスクリプション管理テスト

1. ダッシュボードの「Manage Subscription」をクリック
2. Stripe Customer Portal > 「Cancel subscription」
3. ダッシュボードに「Cancels on [日付]」が表示されることを確認

### 本番環境への移行

1. Stripe Live mode に切り替え
2. 本番用 Product & Price を作成
3. 本番用 Webhook を登録
4. 環境変数を `sk_test_` → `sk_live_` に更新
5. Vercel / Supabase に本番環境変数を設定 > 再デプロイ
