# Lecsy English Promo Video — 10 Patterns

> **共通仕様**
> - Format: 1080×1920 (9:16 vertical / portrait)
> - Duration: 15–25 seconds
> - FPS: 30
> - Engine: Remotion
> - Voiceover: ElevenLabs (English, 1 voice per video — voices must NOT overlap between scenes)
> - Font: Inter (Google Fonts)
> - Background music: None (voiceover only) or very subtle lo-fi pad (< -20dB)
> - Each pattern uses a completely different voiceover script — no reuse across patterns

## Voice Assignment Guide (ElevenLabs — NO OVERLAP)

Each pattern should use a **single voice** that plays one scene at a time. Voiceover for each scene starts AFTER the previous scene's audio has finished. Use these ElevenLabs settings as baseline:

```json
{
  "model_id": "eleven_multilingual_v2",
  "voice_settings": {
    "stability": 0.7,
    "similarity_boost": 0.75,
    "style": 0.3
  }
}
```

**Recommended voice rotation** (to keep content feeling fresh across 10 patterns):
- Patterns 1, 3, 5, 7, 9: Male voice (e.g., "Adam" or "Josh")
- Patterns 2, 4, 6, 8, 10: Female voice (e.g., "Rachel" or "Bella")

> Each scene within a single pattern has its own audio file (`scene1-hook.mp3`, `scene2-problem.mp3`, etc.) placed sequentially with no overlap using Remotion's `<Sequence from={...}>`.

## Technical Notes

- Resolution: 1080×1920 (9:16 vertical)
- All videos use Remotion's `<TransitionSeries>` for scene sequencing
- Voiceover audio files placed in `public/voiceover/patternN/` subdirectories
- Each scene voiceover is a separate mp3 file triggered via `<Sequence from={X}>` + `<Audio>`
- Font: Inter via `@remotion/google-fonts/Inter`
- Animations: `spring()` and `interpolate()` from Remotion
- Export command: `npx remotion render LecsyPromoPatternN out/patternN.mp4`
